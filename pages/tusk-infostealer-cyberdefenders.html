Overview

A blockchain development company detected unusual activity when an employee was redirected to an unfamiliar website while accessing a DAO management platform. Soon after, multiple cryptocurrency wallets linked to the organization were drained. Investigators suspect a malicious tool was used to steal credentials and exfiltrate funds.

Use this note as an analyst reference: understand the concept, identify the relevant telemetry, pivot through evidence, and document the decision path clearly.

Tusk Infostealer (CyberDefenders)

Scenario

A blockchain development company detected unusual activity when an employee was redirected to an unfamiliar website while accessing a DAO management platform. Soon after, multiple cryptocurrency wallets linked to the organization were drained. Investigators suspect a malicious tool was used to steal credentials and exfiltrate funds.

Your task is to analyze the provided intelligence to uncover the attack methods, identify indicators of compromise, and track the threat actor’s infrastructure.

1- In KB, what is the size of the malicious file?

921.36

by searching the hash in VirusTotal and look in details will find the file size

2- What word do the threat actors use in log messages to describe their victims, based on the name of an ancient hunted creature?

Mammoth

searching with the hash i start looking at the reports and find the answer in this report

https://securelist.com/tusk-infostealers-campaign/113367/

3- The threat actor set up a malicious website to mimic a platform designed for creating and managing decentralized autonomous organizations (DAOs) on the MultiversX blockchain (peerme.io). What is the name of the malicious website the attacker created to simulate this platform?

tidyme.io

we can find the answer in the same report (other answers as well)

Decentralized Autonomous Organizations (DAOs) are entities that operate on blockchain networks, relying on smart contracts to automate governance and decision-making without the need for centralized authority. These organizations are typically used for funding projects, managing cryptocurrency treasuries, and facilitating community driven decision making.

in our case we see that the threat actor create a fake website mimicking the legitimate DAO called

peerme.io to trick users to interact with malicious links

4- Which cloud storage service did the campaign operators use to host malware samples for both macOS and Windows OS versions?

Dropbox

in the same report we see the attacker used DropBoxfor storing the malicious payloads

5- The malicious executable contains a configuration file that includes base64-encoded URLs and a password used for archived data decompression, enabling the download of second-stage payloads. What is the password for decompression found in this configuration file?

newfile2024

we can see the configuration file named config.json contains base64 encoded URLs that point to second-stage payloads. also it contains a password for decompressing archived files downloaded from these URLs

The password for decompressing the archived payload is newfile2024.

6- What is the name of the function responsible for retrieving the field archive from the configuration file?

downloadAndExtractArchive

the malware is retrieving additional payloads from external sources (Dropbox link in the Config file)

this happen through a JavaScript downloader which fetch and execute second stage payloads

(preload.js) from the report we can see the 2 functions that handle the downloading and executions of payloads

7- In the third sub-campaign carried out by the operators, the attacker mimicked an AI translator project. What is the name of the legitimate translator, and what is the name of the malicious translator created by the attackers?

Yous.ai, voico.io

in the third sub campaign we see the attackers impersonate and AI translator project to lure victims into downloading malware

now most attackers are exploiting the rise of AI to increase their success rate of their phishing and malware distribution

8- The downloader is tasked with delivering additional malware samples to the victim’s machine, primarily infostealers like StealC and Danabot. What are the IP addresses of the StealC C2 servers used in the campaign?

46.8.238.240,23.94.225.177

ctrl + f then search C2

StealC is a modular infostealer that has gained popularity among cybercriminals due to its flexibility and ability to harvest data from various sources, including browsers, password managers, and cryptocurrency extensions.

9- What is the address of the Ethereum cryptocurrency wallet used in this campaign?

0xaf0362e215Ff4e004F30e785e822F7E20b99723A

ctrl + f then search ETH (Ethereum)

attackers leveraged a malware designed to monitor clipboard activity and replace copied cryptocurrency wallet addresses with those controlled by the attacker.
