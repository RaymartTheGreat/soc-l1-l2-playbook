Original Notes

Stenography

Steganography is the practice of hiding messages or information within ordinary, non-secret data.

For instance, secret text can be embedded inside an image file so that it appears as a normal picture to anyone who views it. With the right tools, however, the hidden content can be extracted. Steganography can also involve placing concealed data in file metadata or appending it beyond the visible end of a file.

Quick workflow

  1. file target — confirm file type / container mismatch.
  2. exiftool target — inspect metadata (comments, software, unusual fields).
  3. strings target | less and hexdump -C target | less — scan for readable payloads and magic bytes (50 4B 03 04 = ZIP, %PDF = PDF, 89 50 4E 47 = PNG).
  4. binwalk -e target — extract embedded files/signatures.
  5. foremost -i target -o outdir or scalpel — carve common file types.
  6. zsteg target.png / stegsolve — check PNG/BMP LSBs and bitplanes.
  7. steghide info target → steghide extract -sf target — check/extract steghide containers.
  8. stegseek target wordlist.txt / stegcracker target rockyou.txt — try common passphrases.
  9. Audio: ffmpeg -i file.mp3 out.wav → open in Sonic Visualiser or Audacity, view spectrogram.
  10. Reverse-image search (Yandex / Google Images / Google Lens) — compare with originals.

Tools

  • exiftool target.jpg
  • file target
  • strings target | less
  • hexdump -C target | less or xxd target | less
  • binwalk -e target
  • foremost -i target -o outdir
  • zsteg target.png
  • stegsolve (GUI) — inspect bitplanes/channels
  • steghide info target.jpg → steghide extract -sf target.jpg
  • stegseek image.jpg wordlist.txt
  • stegcracker image.jpg rockyou.txt
  • wavsteg -h -i sound.wav -s secret.txt -o sound_steg.wav (hide)
  • wavsteg -r -i sound_steg.wav -o out.txt (extract)
  • Brute-force example: python2 steg_brute.py -b -d /path/rockyou.txt -f image.jpg

steghide example

  • Prepare payload: zip -r secret.zip secret_folder
  • Embed: steghide embed -cf cover.jpg -ef secret.zip -sf cover_stego.jpg (enter passphrase)
  • Inspect: steghide info cover_stego.jpg
  • Extract: steghide extract -sf cover_stego.jpg → enter passphrase → get secret.zip.

Audio

  • Convert: ffmpeg -i file.mp3 out.wav
  • Inspect spectrogram in Audacity or Sonic Visualiser (zoom & adjust window/scale).
  • Look for clear patterns, text, or images; check phase/time-domain for repeated tones.
  • Try LSB tools (wavsteg) and check for appended data (hexdump).

Magic bytes to look for

  • ZIP: 50 4B 03 04 (PK..)
  • PNG: 89 50 4E 47
  • PDF: %PDF
  • JPEG: FF D8 FF
  • GIF: 47 49 46 38 (GIF8)
  • BMP: 42 4D (BM)
  • RAR: 52 61 72 21 1A 07 00
  • 7z: 37 7A BC AF 27 1C
  • EXE (PE): 4D 5A (MZ)
  • WAV: 52 49 46 46 (RIFF) + 57 41 56 45 (WAVE)
  • MP3: 49 44 33 (ID3)
