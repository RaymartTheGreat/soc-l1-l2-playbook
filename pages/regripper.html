Search for Specific Keys with RECmd

https://github.com/EricZimmerman/RECmd

powershell
RECmd.exe -f .\NTUSER.dat --sk Discord

--sk Discord :Tells RECmd to look for a subkey named exactly Discord at any level in the registry hierarchy. This does not search values or value data — only key names.

powershell
RECmd.exe -f .\NTUSER.dat --search Zoom

This will recursively search the entire hive for any key names, value names, or value data containing "Zoom" — such as:

  • Software\Zoom
  • InstallPath values pointing to C:\Program Files (x86)\Zoom\Bin
  • MRU (Most Recently Used) entries
  • RecentDocs or UserAssist entries
