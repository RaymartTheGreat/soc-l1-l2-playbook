Original Notes

Attachment Analysis

Phishing emails often deliver malware through attachments. Proper analysis helps identify malicious behavior, understand payloads, and trace attacker intent.

1- Safe Extraction of Attachments

  • Via Email Client GUI:

Right-click the attachment → Save As → Choose analysis folder (never open directly).

  • Using Tools (CLI-based):
    • emldump.py: Extracts attachments from .eml files.

`` emldump.py suspicious.eml -s 3 -d > attachment.doc ``

2- Hashing the File

Hash the attachment to:

  • Check against threat intel databases.
  • Share safely in reports.
  • Block via EDR or email gateway.

Linux/macOS:

bash
sha256sum file
sha1sum file
md5sum file

Windows PowerShell:

powershell
Get-FileHash file.docx              # Default SHA256
Get-FileHash file.docx -Algorithm MD5
Get-FileHash file.docx -Algorithm SHA1

3- Static Analysis

  • Analyze file metadata (e.g., author, timestamps, embedded links).
  • Inspect macros/scripts in Office files (use tools like oletools or olevba).
  • Decompress archives to inspect contents.

4- Dynamic Analysis (Sandboxing)

Run the file in a sandboxed environment to observe real behavior.

Key behaviors to monitor:

  • 🧠 Process Activity:

Spawning of unexpected processes (e.g., cmd.exe, powershell.exe, wscript.exe).

  • 🧬 Registry Activity:

Creation/modification of entries for persistence.

  • 🌐 Network Connections:

Outbound connections to unknown IPs, domains, or C2 servers.

  • 💾 File Activity:

Dropping payloads, modifying system files, or creating startup scripts.

5- Common Malicious Attachment Types

File TypePurpose
.doc, .docm, .xls, .pptmMacro-based malware (Office macros)
.pdfExploits or embedded links
.zip, .rarArchives hiding payloads or loaders
.lnkShortcut to malicious scripts
.js, .vbs, .htaScript-based droppers
.iso, .imgUsed to bypass email filters

6- Threat Intel & Sandboxing Platforms

Use these to check reputation or behavior:

PlatformFunction
VirusTotalFile & hash scan + passive analysis
Cisco TalosReputation, IOC, threat info
Hybrid AnalysisDynamic sandbox reports
Any.RunInteractive sandboxing
Joe SandboxDeep dynamic file behavior

Summary: Attachment Analysis Steps

  1. Extract file safely using email client or tools like emldump.py.
  2. Hash the file (SHA256) and check on VirusTotal, Talos, etc.
  3. Perform static analysis — macros, embedded links, metadata.
  4. Upload to sandbox — monitor network, registry, file/process behavior.
  5. Document indicators of compromise for downstream alerting or blocking.
