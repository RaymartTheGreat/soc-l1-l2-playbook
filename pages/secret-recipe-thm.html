Secret Recipe (THM)

Scenario

Jasmine owns a famous New York coffee shop Coffely

which is famous city-wide for its unique taste. Only Jasmine keeps the original copy of the recipe, and she only keeps it on her work laptop. Last week, James from the IT department was consulted to fix Jasmine's laptop. But it is suspected he may have copied the secret recipes from Jasmine's machine and is keeping them on his machine.

His machine has been confiscated and examined, but no traces could be found. The security department has pulled some important registry artifacts from his device and has tasked you to examine these artifacts and determine the presence of secret files on his machine.

https://tryhackme.com/room/registry4n6

1- How many files are available in the Artifacts folder on the Desktop?

6

2- What is the computer name of the machine found in the registry?

JAMES

SYSTEM\CurrentControlSet\Control\ComputerName \ComputerName

3- When was the Administrator account created on this machine? (Format: yyyy-mm-dd hh:mm:ss)

2021-03-17 14:58:48

for this information we will take a look at SAM hive which contain user and group information

SAM\Domains\Account\Users

4- What is the RID associated with the Administrator account?

500

The Relative Identifier (RID) is a part of a Security Identifier (SID) that uniquely identifies a user or a group within a domain. In Windows, each user and group account has a SID, and the RID is the last portion of that SID.RID refers to the user ID in the SAM (Security Account Manager) database.

5- How many user accounts were observed on this machine?

7

check for the names folder or in down right you see the total number of rows

6- There seems to be a suspicious account created as a backdoor with RID 1013. What is the account name?

bdoor

7- What is the VPN connection this host connected to?

ProtonVPN

for this we will look for NetworkList in Software Hive

SOFTWARE hive which contains information on tools and software that users on the machine used.

8- When was the first VPN connection observed? (Format: YYYY-MM-DD HH:MM:SS)

2022-10-12 19:52:36

9- There were three shared folders observed on his machine. What is the path of the third share?

C:\RESTRICTED FILES

System hive contain information about stuff like shared folders so we search for shares in system

10- What is the last DHCP IP assigned to this host?

172.31.2.197

SYSTEM\CurrentControlSet\Services\Tcpip \Parameters\Interfaces

11- The suspect seems to have accessed a file containing the secret coffee recipe. What is the name of the file?

secret-recipe.pdf

for this will look in NTUser.dat hive which have information on recently accessed files

Recent Files: NTUSER.DAT\Software\Microsoft\Windows \CurrentVersion\Explorer\RecentDoc

12- The suspect executed multiple commands using the Run window. What command was used to enumerate the network interfaces?

pnputil /enum-interfaces

NTUSER.DAT/Software/Microsoft/Windows/CurrentVersion/Explorer

13- The user searched for a network utility tool to transfer files using the file explorer. What is the name of that tool?

netcat

for this we check WordWheelQuery

WordWheelQuery stores the last search terms entered by a user in the Windows Start Menu or File Explorer search box.

14- What is the recent text file opened by the suspect?

secret-code.txt

back to RecentDocs and look in .txt

NTUSER.DAT\Software\Microsoft\Windows \CurrentVersion\Explorer\RecentDocs

15- How many times was PowerShell executed on this host?

3

NTUSER.DAT\Software\Microsoft\Windows\CurrentVersion\Explorer\UserAssist

we need to expand the UserAssist and go through each key until we find one with a count, view to the right and scroll for powershell

16- The suspect also executed a network monitoring tool. What is the name of the tool?

wireshark

in the same view as last question just scroll down and you will see the answer

17- Registry Hives also note the amount of time a process is in focus. Examine the Hives and confirm for how many seconds was ProtonVPN executed?

343

scroll up tell you see ProtonVPN and convert minutes into seconds

18- Everything.exe is a utility used to search for files in a Windows machine. What is the full path from which everything.exe was executed?

C:\Users\Administrator\Downloads\tools\Everything\Everything.exe

scroll down will see a row for everything.exe with the full path
