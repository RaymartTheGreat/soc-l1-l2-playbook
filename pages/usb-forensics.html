USB forensics

This photo was made by my friend Mahmoud Shaker: https://www.linkedin.com/in/mahmoud-shaker-2aa923231/

USB forensics focuses on tracking and analyzing the use of USB devices on a system. It helps determine:

  • Which USB devices were connected
  • Which user connected them
  • When devices were connected and removed
  • Evidence of files opened from the device

User Information and Activity with USB Devices

To investigate user interaction with a USB device, the following can be determined:

  • Drive Letter, Device, and Volume Name – Helps identify the physical device and its mount point.
  • User that used the specific USB device – Determined via registry entries and user profiles.
  • First and Last Time Device Connected – Registry timestamps reveal the initial and most recent usage.
  • Time Device Removed – Some registry keys track device removal events.

USB Device Types

  1. MSC (Mass Storage Class)
    • Appears in Windows Explorer as:
      • Hard Disk Drives
      • Removable Storage Devices
  2. PTP (Picture Transfer Protocol)
    • Appears in Windows Explorer as:
      • Windows Portable Devices (WPD)
      • Mounted as portable devices
  3. MTP (Media Transfer Protocol) – Extension of PTP
    • Devices include MP3 players, cameras, smartphones, and tablets

Evidence of File Opening

MSC Devices

  • Creates LNK files for files opened directly from the USB device.
  • Key directories to check:
    1. C:\Users\<username>\AppData\Roaming\Microsoft\Windows\Recent
      • Contains LNK files for all directly opened files.
    2. C:\Users\<username>\AppData\Roaming\Microsoft\Windows\Recent\AutomaticDestinations
      • Stores jumplist files.
    3. C:\Users\<username>\AppData\Roaming\Microsoft\Office\Recent
      • Contains LNK files for Office documents only.

MTP Devices

  • LNK files may or may not be created.
  • Key evidence locations:
    1. Office documents:

C:\Users\<username>\AppData\Roaming\Microsoft\Office\Recent

      • Two LNK files per document (one for file, one for folder)
    • JPG files:
      • True LNK files created pointing back to the device.
    • Temporary copies:

C:\Users\<username>\AppData\Local\Temp\WPDNSE\{FOLDERGUID}

      • Maintains temporary copies of opened files (does not survive reboot)

Registry Locations for USB Analysis

USBSTOR

  • Tracks MSC devices plugged into the system.
  • Contains Vendor, Product, Version, and connection timestamps.
  • Location:

`` HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\USBSTOR ``

Volume Name Discovery

  • Location:

`` HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Portable Devices\Devices ``

Drive Letters and GUIDs

  • Track drive letters, physical drives, and external enclosures:

`` HKEY_LOCAL_MACHINE\SYSTEM\MountedDevices ``

Volume Name and USB Serial

  • Location for converting serial numbers to determine connections:

`` HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\EMDMgmt ``

First, Last Connected, and USB Removal

  • Location:

`` SYSTEM\CurrentControlSet\Enum\USBSTOR\Ven_Prod_Version\USB iSerial #\Properties\{83da6326-97a6-4088-9453-a1923f573b29}\#### ``

SetupAPI Log

  • Tracks device installation and driver information:

`` C:\Windows\inf\setupapi.dev.log ``
