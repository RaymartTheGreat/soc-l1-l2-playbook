Browser Forensics (BTLO)

Scenario

Our SOC alerted that there is some traffic related to crypto mining from a PC that was just joined to the network. The incident response team acted immediately, observed that the traffic is originating from browser applications. After collecting all key browser data using FTK Imager, it is your job to use the ad1 file to investigate the crypto mining activity.

https://blueteamlabs.online/home/challenge/browser-forensics-cryptominer-aa00f593cb

Note : Every Chrome extension must include a manifest.json file in its root directory. The manifest lists the extension’s name, version, permissions, background scripts, content scripts and other important metadata / it’s the first file you inspect when hunting for malicious extensions, because it shows which files the extension will load and what privileges it requests. Think of manifest.json as the extension’s table of contents: it tells you what to open next.

1- How many browser-profiles are present in Google Chrome?

2

I searched for the Chrome profile location and opened AppData\Local\Google\Chrome\User Data. There were two profile folders (e.g., Default and Profile 1)

2- What is the name of the browser theme installed on Google Chrome?

Earth in Space

I checked installed extensions/themes by exporting the Chrome user directory and reviewing the extension directories. I also used BrowsingHistoryView to confirm the user had searched for “Google Chrome themes” and then opened the Chrome Web Store and downloaded the theme. In FTK Imager I opened the theme’s manifest.json inside the theme extension folder and confirmed the name field was Earth in Space

3- Identify the Extension ID and Extension Name of the cryptominer

egnfmleidkolminhjlkaomjefheafbbb, DFP Cryptocurrency Miner

I iterated through each extension folder and inspected manifest.json files looking for suspicious background scripts, remote script URLs, or permissions like "tabs", "webRequest", or "nativeMessaging". The extension with ID egnfmleidkolminhjlkaomjefheafbbb had a description claiming it “allows staff members to mine cryptocurrency in the background of their web browser.” A quick Google search for that ID / name shows the extension was removed from the Chrome Web Store for malware this corroborates the find.

Theme path :
C:\Users\<USERNAME>\AppData\Local\Google\Chrome\User Data\Default\Preferences

4- What is the description text of this extension?

Allows staff members to mine cryptocurrency in the background of their web browser

from the previous question

5- What is the name of the specific javascript web miner used in the browser extension?

cryptoloot

I exported the background.js, renamed it to a non-executable extension (.noextension) and opened it in Notepad++. Inside background.js I found the cryptominer code and configuration (the mining library reference, the site/public key, and the worker identifiers)

6- How many hashes is the crypto miner calculating per second?

20

7- What is the public key associated with this mining activity?

b23efb4650150d5bc5b2de6f05267272cada06d985a0

8- What is the URL of the official Twitter page of the javascript web miner?

twitter.com/CryptoLootMiner

simple google search
