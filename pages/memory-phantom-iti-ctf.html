I found a base64 string: mU6emFmmsHe+rWert2e9unGrgg==, but it's still not the answer.

So, I grep for DecryptFlag that I found in the message it told me to check.

External image reference: https://miro.medium.com/v2/resize:fit:700/10AF3nvE-a6TtnoCszCVqQ.png

Now I found an XOR key: xor_key = 0xff22ff.

#XOR is a bitwise operation used in cryptography to modify data using a key. The key here is 0xff22ff, which is applied repeatedly to the data.

Now, I created this Python script to retrieve the flag.

python
import base64

# Input Base64 string and XOR key
encoded_data = "mU6emFmmsHe+rWert2e9unGrgg=="  # Base64 string
xor_key = bytes([0xFF, 0x22, 0xFF])  # XOR key represented as bytes

# Step 1: Decode the Base64 string
decoded_data = base64.b64decode(encoded_data)
print("Decoded Base64 Data:", decoded_data)

# Step 2: XOR the decoded data with the XOR key
# XOR each byte of the decoded data with the key, cycling through the key
xor_result = bytes([byte ^ xor_key[i % len(xor_key)] for i, byte in enumerate(decoded_data)])
print("XOR Decoded Result:", xor_result)

External image reference: https://miro.medium.com/v2/resize:fit:700/1VEZo7UC5hcLtCBPArc1A.png

That’s it! We got the flag: flag{YOUARETHEBEST}

There’s a simpler way than my method - you can use CyberChef.

  1. Use the “From Base64” operation to decode the input.
  2. Add the “XOR” operation and set the XOR key (FF22FF).
