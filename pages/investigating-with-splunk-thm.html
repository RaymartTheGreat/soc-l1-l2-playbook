Investigating with Splunk (THM)

Scenario

SOC Analyst Johny has observed some anomalous behaviours in the logs of a few windows machines. It looks like the adversary has access to some of these machines and successfully created some backdoor. His manager has asked him to pull those logs from suspected hosts and ingest them into Splunk for quick investigation. Our task as SOC Analyst is to examine the logs and identify the anomalies.

https://tryhackme.com/room/investigatingwithsplunk

1- How many events were collected and Ingested in the index main?

12256

2- On one of the infected hosts, the adversary was successful in creating a backdoor user. What is the new username?

A1berto

I filtered with event code 4720 (user account was created) and got only one event

3- On the same host, a registry key was also updated regarding the new backdoor user. What is the full path of that registry key?

HKLM\SAM\SAM\Domains\Account\Users\Names\A1berto

filter with event code 12 (Registry object added/modified) and event code 13 (Detects renaming of registry keys/values) with filtering with the backdoor user

4- Examine the logs and identify the user that the adversary was trying to impersonate.

Alberto

5- What is the command used to add a backdoor user from a remote computer?

C:\windows\System32\Wbem\WMIC.exe" /node:WORKSTATION6 process call create "net user /add A1berto paw0rd1

filter with event creation event codes ( 1 , 4688 ) and we can a remote account creation using WMIC

6- How many times was the login attempt from the backdoor user observed during the investigation?

0

filter with successful and failed logon and we got 0

7- What is the name of the infected host on which suspicious Powershell commands were executed?

James.browne

filter with event code 4104 ( logs PowerShell script block execution, capturing the full content) and event code 4103 (logs module logging events, recording executed cmdlets from specific PowerShell modules)

8- PowerShell logging is enabled on this device. How many events were logged for the malicious PowerShell execution?

79

9- An encoded Powershell script from the infected host initiated a web request. What is the full URL?

hxxp[://]10[.]10[.]10[.]5/news[.]php

using cyberchef we decode the powershell using from base64 and decode text (UTF-16LE (1200) )

recipes . the decoded script is used to modify the powershell script block logging setting additionally we see another base 64 encoded value that refer the domain with /news.php
