Disk Analysis & Autopsy (THM)

In the attached VM, there is an Autopsy case file and its corresponding disk image. After loading the .aut file, make sure to re-point Autopsy to the disk image file.

External image reference: https://assets.tryhackme.com/additional/autopsy/autopsy-fix-missing-image.png

Ingest Modules were already ran for your convenience.

Your task is to perform a manual analysis of the artifacts discovered by Autopsy to answer the questions below.

https://tryhackme.com/room/autopsy2ze0

1- What is the MD5 hash of the E01 image?

3f08c518adb3b5c1359849657a9b2079

we can find the hash of the image by selecting appropriate data source and open container tab

2- What is the computer account name?

DESKTOP-0R59DJ3

we can find the answer under the operating system information

3- List all the user accounts. (alphabetical order)

H4S4N,joshwa,keshav,sandhya,shreya,sivapriya,srini,suba

in operating system user account will see list of all uses he is asking about user accounts so will ignore Guest, LocalService, etc or we look at the users starting with UID of 1000

4- Who was the last user to log into the computer?

sivapriya

sort by data date accessed

5- What was the IP address of the computer?

192.168.130.216

in program files you can see a tool for network monitoring open the file inside it can see it contains network information of the system (LANIP conotains ip of the system)

6- What was the MAC address of the computer? (XX-XX-XX-XX-XX-XX)

08-00-27-2c-c4-b9

in the same location can see the mac address in “LANNIC” copy it then edit its format

7- What is the name of the network card on this computer?

Intel(R) PRO/1000 MT Desktop Adapter

after searching with networkcards we see the registry path to it after going to the location we can find a subkey called 2 inside it can find the answer

8- What is the name of the network monitoring tool?

Look@LAN

already seen it in question 5

9- A user bookmarked a Google Maps location. What are the coordinates of the location?

12°52'23.0"N 80°13'25.0"E

in extracted content will find web bookmarks

10- A user has his full name printed on his desktop wallpaper. What is the user's full name?

Anto Joshwa

open NTUSER.dat and look in Desktop

but what i did is going through every user but lucky me find it in the second one in the downloads folder expand the window will see the full name

11- A user had a file on her desktop. It had a flag but she changed the flag using PowerShell. What was the first flag?

flag{HarleyQuinnForQueen}

checking the powershell history for each user in Shreya histor we can see the first flag was i hacked you then changed to harleyquinnforqueen

12- The same user found an exploit to escalate privileges on the computer. What was the message to the device owner?

flag{I-hacked-you}

in the same folder take a look at the powershell script will see a text file will be created after the execution and the text file is the flag

13- 2 hack tools focused on passwords were found in the system. What are the names of these tools? (alphabetical order)

Lazagne,Mimikatz

the first tool we see located at H$S$N downloads folder finding the second one takes a while but then i got the idea its a hack tools so checked windows defenders alerts going through files in this directory we can see alerts for both Mimikatz and Lazagne

14- There is a YARA file on the computer. Inspect the file. What is the name of the author?

Benjamin DELPY (gentilkiwi)

filtering with .yar then click the file we found or suspect by clicking of the file we can see the answer

15- One of the users wanted to exploit a domain controller with an MS-NRPC based exploit. What is the filename of the archive that you found? (include the spaces in your answer)

2.2.0 20200918 Zerologon encrypted.zip

https://www.crowdstrike.com/en-us/blog/cve-2020-1472-zerologon-security-advisory/

searched with MS-NRPC then find the exploit is also known as zerologon and sp i filter with it and got the answer
