Notes/Tips

  • windows.info plugin : gets general information about the Windows operating system, such as version, build number, system root, number of processors, and product type.
  • windows.netscan plugin : scans memory for network connection structures and displays active TCP and UDP connections, including local/remote IPs, ports, connection states, and owning process. Useful for identifying suspicious or unexpected network activity.
  • windows.pstree plugin : shows the process list in a tree format based on parent-child relationships, including process names, PIDs, and creation times. Helps visualize process hierarchy and spot suspicious or orphaned processes.
  • windows.filescan plugin : scans memory for FILEOBJECTs and lists file handles opened by processes, including file paths, offsets, and file names. Useful for discovering hidden or recently accessed files, even if they were deleted or not listed in standard file structures.
  • windows.mftscan.MFTScan plugin – scans memory for raw entries from the NTFS Master File Table (MFT), showing file metadata like name, timestamps, and flags. Useful for identifying deleted or hidden files and reconstructing file system activity.
