Scenario

Unusual network activity has been detected within a university environment, indicating potential malicious intent. These anomalies, observed six hours ago, suggest the presence of command and control (C2) communications and other harmful behaviors within the network.

Your team has been tasked with analyzing recent network traffic logs to investigate the scope and impact of these activities. The investigation aims to identify command and control servers and uncover malicious interactions.

https://cyberdefenders.org/blueteam-ctf-challenges/nerisbot/

Initial Unauthorized Access

Attacker's IP Address: 195.88.191.59

Detection Method:

jsx
index=* sourcetype=suricata eventtype=suricata_eve_ids_attack
| stats values(dest_ip) values(http.hostname) by src_ip

This query aggregates destination IPs and hostnames associated with each source IP, helping to identify potential attacker IP addresses.

Attacker's Domain Name

Domain Name: nocomcom.com

Detection Method:

jsx
index=* sourcetype=suricata event_type=dns "dns.answers{}.rdata"="195.88.191.59"
| stats count by dns.rrname

This query identifies DNS queries that resolved to the attacker's IP address, revealing the associated domain name.

Targeted System's IP Address

Victim's IP Address: 147.32.84.165

Detection Method:

jsx
index=* sourcetype=suricata dest_ip=195.88.191.59 http.url=*
| table _time src_ip dest_ip http.url
| sort + _time

This query filters HTTP requests to the attacker's IP, identifying the source IPs (victims) accessing malicious URLs.

Downloaded Files on Compromised Host

Unique Files Downloaded: 3

Potentially Malicious Files: 2

Detection Method:

Identify files downloaded from the attacker's server.

jsx
index=* sourcetype=suricata src_ip=147.32.84.165 dest_ip=195.88.191.59 url=*
| stats count by http.url

This query counts unique URLs accessed by the victim, indicating the number of files downloaded.
