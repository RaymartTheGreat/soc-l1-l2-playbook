User Activity Analysis

1- ShellBags

Registry Locations

matlab
HKCU:\Software\Microsoft\Windows\Shell\BagMRU
HKCU:\Software\Microsoft\Windows\Shell\Bags

Hive Files

  • %USERPROFILE%\NTUSER.DAT
  • %USERPROFILE%\AppData\Local\Microsoft\Windows\UsrClass.dat

Overview

ShellBags record metadata about folder views and paths accessed by the user persisting details even for deleted or disconnected folders.

Key Data Extracted

  • Folder View Settings: View mode (e.g., details, icons), window size, icon positions
  • Folder Paths: Paths to local, network, and external-drive directories
  • Timestamps: First created, last accessed, last written
  • User Preferences: Sort order, column widths, window position
  • Deleted/Disconnected Folders: Evidence of folders no longer present on the system

Forensic Value

  • Tracks user navigation and view preferences
  • Reveals access to external or network storage
  • Supports timeline reconstruction even after file/folder deletion

Tool Example:

  • ShellBags Explorer (sbecmd.exe)
    1. Run as Administrator.
    2. Load NTUSER.DAT and UsrClass.dat to enumerate all BagMRU/Bags entries.

2- Typed Paths (Live Location)

Registry Path

matlab
HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\TypedPaths

Description

Stores the most recently typed entries in the Run dialog or File Explorer address bar.

Forensic Value

  • Reveals explicit user navigation intent
  • Identifies commands, UNC paths, or URLs manually entered by the user

3- WordWheel Query (Live Location)

Registry Path

matlab
HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\WordWheelQuery

Description

Records search terms entered into File Explorer’s search box.

Forensic Value

  • Shows keywords or filenames sought by the user
  • Uncovers hidden investigative queries or search patterns

4- RecentDocs

Registry Path

matlab
HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\RecentDocs

Description

Lists recently opened documents (files of any extension).

Forensic Value

  • Identifies documents accessed within user sessions
  • Aids in locating potentially incriminating files

5- Common Dialog (ComDlg32)

5-1 LastVisitedMRU

matlab
HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\ComDlg32\LastVisitedMRU

Tracks folders navigated via the Open/Save dialog.

5-2 LastSavedPidlMRU

matlab
HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\ComDlg32\LastSavedPidlMRU

Records locations where files were most recently saved or opened.

Forensic Value

  • Maps user’s file navigation and save operations
  • Complements RecentDocs by showing dialog-based access paths

6- UserAssist

Registry Path

matlab
HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\UserAssist

Contains two GUID subkeys:

  • {CEBFF5CD-ACE2-4F4F-9178-9926F41749EA}: Explorer-based file/folder launches
  • {F4E57C4B-2036-45F0-A9AB-443BCFE33D9F}: Shortcut-based (LNK) executions

Data Recorded

  • Application/file launch count
  • Last execution timestamp (ROT13-encoded values)

Forensic Value

  • Highlights frequently used applications and tools
  • Detects anomalous or unauthorized program executions

7- RunMRU

Registry Path

matlab
HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\RunMRU

Description

Lists the most recent commands or executables run via the Run dialog.

Forensic Value

  • Reveals direct command-line or tool execution attempts.
  • Useful for identifying one-off commands or malware launched manually.

8- Shortcut Files (.LNK)

Locations

python
%USERPROFILE%\AppData\Roaming\Microsoft\Windows\Recent
%USERPROFILE%\Recent

Description

Shortcuts created when a user opens a file or application. Examined using LECmd.exe or similar.

Key Attributes

  • Target path and file name
  • Access timestamps
  • File size and network share details

Forensic Value

  • Pinpoints exact file and location accessed
  • Records network vs. local access

Analyzing a LNK file using LECmd.exe

9- Jump Lists

Paths

python
%APPDATA%\Microsoft\Windows\Recent\AutomaticDestinations
%APPDATA%\Microsoft\Windows\Recent\CustomDestinations

Description

Jump Lists store recently or frequently accessed items per application, documents, websites, or tasks.

Forensic Value

  • Diary of interactive sessions: which files/apps were used, access counts, timestamps.
  • Records documents opened, network locations accessed, and tasks launched.

Analysis Tools

  • JumpListExplorer.exe
    1. Load .automaticDestinations-ms or .customDestinations-ms files.
    2. View list entries, access counts, file paths, and timestamps.
