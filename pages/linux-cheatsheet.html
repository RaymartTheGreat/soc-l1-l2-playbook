Linux CheatSheet

Word and Line Counting

  • wc : Display word, line, and byte counts.
  • wc -l : Count lines.
  • wc -w : Count words.
  • wc -c : Count bytes.

File Head and Tail

  • head : Display the first 10 lines.
  • head -n : Display the first n lines.
  • tail : Display the last 10 lines.
  • tail -n : Display the last n lines.

Text Filtering and Sorting

  • cut file -d "" -f 1 : Extract the first field using a delimiter.
  • sort : Sort lines alphabetically or numerically.
  • sort -nr : Sort numerically in reverse order.
  • uniq :Remove duplicate lines.
  • uniq -c : Count and display duplicate lines.

Search and Pattern Matching

  • grep -v : Exclude lines matching a pattern.
  • grep -i : Case-insensitive search.
  • grep -n : Show line numbers of matches.
  • grep -B : Show n lines before a match.
  • grep -A : Show n lines after a match.
  • grep -C : Show n lines of context around a match.
  • grep -u : Remove duplicates
  • grep -r : searches recursively through all files in the current directory and its subdirectories for lines containing the word
  • grep -P '(?=.value1)(?=.value2)' : Match lines containing both value1 and value2 (any order; requires grep -P for Perl regex).
  • grep -E 'value1|value2' : Match lines containing either value1 or value2 (OR).
  • grep 'x' file | grep 'y' : Match lines containing both x and y by piping two grep commands (order doesn't matter).

Cutting and Extracting Data

  • cut -d 'delimiter' -f field_number: Extract specific field(s) from a delimited file.
  • cut -d ' ' -f 1: Extract the first field (space-delimited).
  • cut -c position_range: Extract characters by position (e.g., cut -c 1-5 for first 5 characters).
  • cut -f field_number: Extract fields when the delimiter is tab (default delimiter).
  • cut -d ',' -f 1,3: Extract the 1st and 3rd fields using a comma as the delimiter.

Scripting and JSON Filtering

  • awk : Pattern scanning and text processing.
  • select (.event.PROCESS_ID == 200) logs.json : Filter JSON for events with PROCESS_ID 200.

File Searching and Processing

  • find : Search for files and directories.
  • find /path -name "file.txt" : Search for a file by name in a specific path.
  • find /path -type d : Search for directories only.
  • find /path -size +1M : Search for files larger than 1MB.
  • grep "pattern" file.txt : Search for a pattern in a file.
  • grep -r "pattern" /path : Search recursively in a directory.
  • sed -n '10p' : Print the 10th line of a file.
  • sed -n '10,15p' : Print lines 10 to 15 of a file.
  • sed 's/old/new/g' file.txt : Replace all occurrences of "old" with "new" in a file.
  • awk 'NR < 11 {print $0}' : Print the first 10 lines of a file.
  • awk '{print $1, $3}' file.txt : Print the first and third columns of a file.
  • sort file.txt : Sort lines in a file alphabetically.
  • sort -r file.txt : Sort lines in reverse order.
  • uniq file.txt : Remove duplicate lines in a file.
  • uniq -c file.txt : Count duplicate lines in a file.
  • cut -d',' -f1 file.txt : Extract the first field from a comma-separated file.
  • xargs command : Build and execute commands from standard input.

Filesystem Navigation

  • ls : List directory contents.
  • ls -a : Show all files, including hidden files.
  • ls -h : Show file sizes in human-readable format.
  • ls -R : Recursively list all subdirectories.
  • *ls -d / : Show only directories.
  • ls -1 : List one file per line.
  • cd : Change the current directory.
  • pwd : Print the current directory path.
  • mkdir : Create a new directory.
  • rm -r : Remove directories and their contents.
  • cp : Copy files or directories.
  • mv : Move or rename files or directories.
  • history : Show the list of recently executed commands.

File Permissions

  • chmod : Change file permissions.
  • chown : Change file ownership.
  • ls -l : View file permissions and ownership.

Networking and System Monitoring

  • ping : Check connectivity to a host.
  • netstat -tuln : List open ports and listening services.
  • ifconfig : Display network interface information.
  • top : Monitor system processes in real-time.
  • htop : Interactive process monitor (if installed).
  • df -h : Show disk usage in human-readable format.
  • du -sh : Display the size of a directory.
  • traceroute : Trace the route packets take to a host.

Process Management

  • ps aux : View running processes.
  • kill -9 [PID] : Force terminate a process.
  • jobs : List background jobs.
  • fg : Bring a background job to the foreground.
  • hash-identifier : Identify hash types

Package Management

  • apt update : Update package lists (Debian/Ubuntu).
  • apt upgrade : Upgrade installed packages.
  • yum install [package] : Install a package (CentOS/RHEL).
  • dnf remove [package] : Remove a package (CentOS/RHEL).

User and Identity Management

  • whoami : Show the current logged-in user.
  • su : Switch user.
  • useradd : Add a new user.
  • userdel : Delete a user.
  • passwd user : Change the password for a user.

Creating and Writing to Files

  • touch filename : Create an empty file (e.g., touch file.txt).
  • echo "text" > file.txt : Create a file and write a line of text to it (overwrites if it exists).
  • echo "text" >> file.txt : Append a line of text to an existing file.
  • cat > file.txt : Create a file and type text into it manually (press Ctrl+D to save).
  • nano file.txt : Create or edit a file using a text editor.
  • > file.txt : Create an empty file or truncate an existing file.

Linux Modules Extension

du – Estimate disk usage of files and directories.

Useful for inspecting space usage recursively.

  • du -sh /var/log : Show total size of /var/log in human-readable format.
  • du -a --time : Include all files and show last modification times.

egrep / fgrep – Variants of grep.

  • egrep supports extended regex.
  • fgrep matches fixed strings (no regex interpretation).
  • egrep "error|fail" logfile.txt : Match either “error” or “fail”.
  • fgrep "GET /index.html" access.log : Match this exact line.

tr – Translate or delete characters from input.

  • tr 'A-Z' 'a-z' : Convert uppercase to lowercase.
  • cat file | tr -cd '[:alnum:] \n' : Remove non-alphanumeric characters.

awk – Field-based pattern scanning and reporting.

  • awk '{print $1}' file.txt : Print first word of each line.
  • awk 'BEGIN { FS="," } {print $2}' : Use comma as field separator.

sed – Stream editor for transforming text.

  • sed 's/old/new/g' file.txt : Replace all “old” with “new”.
  • sed -n '3,5p' file.txt : Print only lines 3 to 5.

xargs – Build command arguments from standard input.

  • find . -name '*.log' | xargs rm : Delete all .log files found.
  • cat list.txt | xargs -n 1 echo : Print each line separately.

curl – Transfer data from or to a server using protocols like HTTP, FTP.

  • curl -O http://example.com/file.txt : Download file with original name.
  • curl -I http://example.com : Show headers only (no body).

wget – Command-line file downloader.

  • wget http://example.com/file.txt : Download file.
  • wget -r --no-parent http://example.com/folder/ : Recursive download.

xxd – Create or reverse hex dumps.

  • xxd file.bin : View contents as hex and ASCII side-by-side.
  • xxd -r hexfile.txt > binary.out : Convert hex back to binary.

gpg – GNU Privacy Guard, for encryption and digital signatures.

  • gpg -c file.txt : Encrypt a file with a passphrase.
  • gpg file.txt.gpg : Decrypt an encrypted file.

tar – Archive files.

  • tar -cvf archive.tar dir/ : Create .tar archive of directory.
  • tar -xvf archive.tar : Extract contents.

id / pwd / uname – Identity and system introspection.

  • id : Show UID, GID, and group memberships.
  • pwd : Print current working directory.
  • uname -a : Print system name, kernel version, architecture.

ps / kill – Process listing and termination.

  • ps aux : Show all running processes with details.
  • kill -9 1234 : Forcefully terminate process with PID 1234.
  • ps -ef | grep nginx : Search for a specific process.

netstat – View active network connections and listening services.

  • netstat -tuln : Show listening TCP/UDP ports numerically.
  • netstat -natp : Show active TCP connections with PID/program names.

less / more – View file contents page-by-page.

  • less file.txt : Scroll through a large file (supports backward search).
  • more file.txt : Paginate through file, but no reverse search.

diff / tee – Compare files or duplicate output.

  • diff file1.txt file2.txt : Show line differences between two files.
  • command | tee output.txt : Write output to both terminal and a file.

file / stat – Determine file type or metadata.

  • file filename : Output file type (text, binary, etc).
  • stat filename : Show timestamps (access, modify, change), permissions, and inode info.

export – Set or modify environment variables.

  • export PATH=$PATH:/new/path : Temporarily add directory to system path.
  • export VAR=value : Set shell variable for current session.

reset – Reinitialize terminal settings.

  • reset : Clear terminal and restore display settings (useful if the screen breaks due to binary dump or bad characters).

systemctl / service – Manage services and daemons.

  • systemctl status ssh : Check if SSH service is running (systemd systems).
  • systemctl start apache2 / stop / restart / enable
  • service nginx status : For SysVinit systems (older distros).
