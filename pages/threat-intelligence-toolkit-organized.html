Threat Intelligence Toolkit (Organized)

📚 Threat Intelligence Knowledge Bases

MITRE ATT&amp;CK

A structured knowledge base of adversary tactics, techniques, and procedures (TTPs) used in real-world attacks. Commonly referenced for threat modeling, detection engineering, and security assessments.

MITRE D3FEND

A defensive counterpart to ATT&CK that catalogs countermeasures mapped to adversary techniques. Useful for blue teamers planning security control implementation and mitigation strategies.

Malpedia

An open repository of malware family information maintained by Fraunhofer Institute. Includes behavioral profiles, YARA rules, threat actor links, and campaign details.

🧠 IOC Aggregation & Threat Sharing Platforms

Open Threat Exchange (OTX)

A collaborative platform by AlienVault for sharing IOCs, attack patterns, and pulse intelligence. Allows subscriptions to community-generated threat feeds.

ThreatFox

A platform for submitting and retrieving malware-related IOCs (IPs, URLs, domains). Offers enrichment and tagging to support real-time detection and feed integration.

IOC Bucket

A free, open-source aggregator of threat intelligence with structured IOCs collected from various community sources. Supports API-based access and download in STIX format.

APT_REPORT

A curated collection of APT reports and related indicators categorized by threat group and geography. Useful for understanding historical APT behaviors and campaigns.

🧪 Malware Analysis & Sandboxing Platforms

ANY.RUN

An interactive sandbox that lets analysts detonate and visually interact with malware in real time. Displays process tree, MITRE mapping, network behavior, and file system changes.

Triage

A cloud sandbox from Hatching. Provides automated malware detonation with behavioral logging, memory dumps, MITRE ATT&CK mapping, and IOC extraction.

CAPEv2

An open-source sandbox framework tailored for unpacking and analyzing malware droppers and evasive samples. Built on Cuckoo, with a focus on extracting configs and payloads.

Joe Security

Dynamic malware analysis system offering full behavioral reporting. Supports Windows, Android, and document exploits.

Hybrid Analysis

Free sandbox for static and dynamic malware analysis. Accepts PE files, scripts, documents, etc. Provides threat scores and YARA rule matching.

MalwareBazaar

Community-powered malware sharing platform. Hosts verified malware samples and integrates with VirusTotal, MISP, and sandbox tools for enrichment.

🧩 Hash & Binary Analysis

VirusTotal

Scans files, URLs, and hashes against 70+ antivirus engines and sandbox tools. Offers historical data, behavior reports, and API integrations.

Malpedia

Includes YARA rules and binary family mapping — useful for classifying samples and identifying actor toolkits.

Gary Kessler's File Signatures

Reference guide for identifying file types based on magic numbers and hex patterns. Helpful in forensic file identification.

CrackStation

Cracks hashed passwords using large precomputed hash tables. Useful during password auditing or credential stuffing investigations.

🛰️ Infrastructure & Reconnaissance

Shodan

Search engine for internet-connected devices. Exposes exposed services, vulnerable systems, and banners — critical for mapping attack surfaces and identifying rogue assets.

SecurityTrails

Comprehensive platform offering DNS, WHOIS, IP, and historical infrastructure insights. Ideal for footprinting, subdomain discovery, and monitoring adversary-owned infrastructure.

BinaryEdge

Performs global internet scans to track exposed systems, open ports, banners, and vulnerabilities. Used for both red teaming and threat detection.

Robtex

Performs deep DNS and routing lookups. Useful for identifying domain-IP relationships, shared infrastructure, and passive DNS insights.

IP Geolocation

Geolocates IPs and enriches them with ASN, ISP, and physical location. Helps correlate IOCs with regions or threat actors.

WHOIS

Retrieves domain registration info, including contact details, creation/expiration dates, and registrar history — key for attribution and timeline building.

CRT.sh

Search engine for Certificate Transparency logs. Reveals issued SSL/TLS certificates for domains — often used for discovering subdomains or cloned phishing infrastructure.

Internet Archive (Wayback Machine)

View historical snapshots of websites. Useful for analyzing pre-compromise states or identifying defaced/phishing sites long after takedown.

MXToolbox

Comprehensive DNS and email infrastructure checker. Investigate SPF, DMARC, MX, and DNS health for potential abuse or misconfigurations.

Reverse DNS Lookup

Identifies PTR records and maps IPs back to domain names — useful for passive recon and IP context validation.

✉️ Phishing, Email & OSINT Enrichment

PhishTank

Community-driven phishing URL database. Allows submission and verification of phishing domains to improve detection and blocking.

URLHaus

Frequent-feed of known malicious URLs, managed by abuse.ch. Excellent source of live IOCs tied to malware campaigns and phishing kits.

Abuse.ch

Ecosystem of threat intel projects (ThreatFox, MalwareBazaar, URLHaus). Offers feeds for botnets, malware, and indicators of compromise.

Have I Been Pwned

Checks email addresses against known data breaches. Helps identify credential reuse, leaked usernames, and password exposures.

Dehashed

Advanced breach database with API access. Searchable by name, email, password, IP, VINs, and more. Useful for deep exposure analysis.

Snusbase

Dark web breach aggregation platform. Allows credential search across leaks, forums, pastes, and private breaches.

IntelX (Intelligence X)

Search engine for darknet content, leaks, public records, and historical snapshots. Commonly used for breach hunting and OSINT investigations.

IOC Bucket

Community-driven repository of structured threat indicators (IP, hash, domain, URL). Useful for threat hunting and enrichment pipelines.

🔐 Breach & Credential Intelligence

Have I Been Pwned

Checks if email addresses or usernames appear in known data breaches. Offers password exposure search and breach alerts.

DeHashed

Breach database search engine that supports queries for emails, usernames, passwords, domains, phone numbers, and IPs.

Snusbase

Aggregates leaked data from forums, paste sites, and breaches. Offers bulk lookups, alerts, and structured breach intelligence.

IntelX (Intelligence X)

OSINT search engine indexing dark web content, pastebins, breached databases, and government leaks. Supports pivoting across multiple identifiers.

BlacklistMaster

Blacklist monitoring and alerting system for IPs and domains. Integrates phishing indicators and threat feeds with detailed scoring and API support.

CrackStation

Password hash cracking utility using massive precomputed tables. Useful for validating leaked credentials or auditing password strength.
