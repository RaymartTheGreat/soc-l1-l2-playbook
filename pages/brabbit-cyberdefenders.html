Original Notes

BRabbit (CyberDefenders)

Scenario

You are an investigator assigned to assist Drumbo, a company that recently fell victim to a ransomware attack. The attack began when an employee received an email that appeared to be from the boss. It featured the company’s logo and a familiar email address. Believing the email was legitimate, the employee opened the attachment, which compromised the system and deployed ransomware, encrypting sensitive files. Your task is to investigate and analyze the artifacts to uncover information about the attacker.

https://cyberdefenders.org/blueteam-ctf-challenges/brabbit/

1- The phishing email used to deliver the malicious attachment showed several indicators of a potential social engineering attempt. Recognizing these indicators can help identify similar threats in the future.

What is the suspicious email address that sent the attachment? theceojamessmith@Drurnbo.com

we can see the attacker use typosquatting technique to trick the victim also there is urgency in the email

2- **The ransomware was identified as part of a known malware family. Determining its family name can provide critical insights into its behavior and remediation strategies.

What is the family name of the ransomware identified during the investigation?**

BadRabbit

using a script to pull the file hash and searched with it in VirusTotal and found that this is BadRabbit ransomware

also searched through other resources that could me understand the ransomware better and help answer the next questions

https://www.virustotal.com/gui/file/630325cac09ac3fab908f903e3b00d0dadd5fdaa0875ed8496fcbb97a558d0da

https://app.any.run/tasks/b1f2e2d8-790a-43c7-b7d3-8f27df9561cf

https://securelist.com/bad-rabbit-ransomware/82851/

https://attack.mitre.org/software/S0606/

3- Upon execution, the ransomware dropped a file onto the compromised system to initiate its payload. Identifying this file is essential for understanding its infection process.

What is the name of the first file dropped by the ransomware?

infpub.dat

by observing the behavior of the malware in any.run sandbox we can see infpub.datwas dropped immediately after execution

This file typically contains encoded data or instructions that facilitate the ransomware's functionality, such as encryption routines or system reconnaissance tasks.

4- Inside the dropped file, the malware contained hardcoded artifacts, including usernames and passwords that could provide clues about its origins or configuration.

What is the only person's username found within the dropped file?

alex

when searching with the hash i come to this report which have the analysis of infpub.dat

https://securelist.com/bad-rabbit-ransomware/82851/

it revealed that it contained hardcoded list of usernames but the only real name is alex

5- After execution, the ransomware communicated with a C2 server. Recognizing its communication techniques can assist in mitigation.

What MITRE ATT&CK sub-technique describes the ransomware’s use of web protocols for sending and receiving data?

T1071.001

you can find the mitre techniques used in VirusTotal in BEHAVIOR→ Activity Summary

we can see that malware used  T1071.001 which uses common application layer protocols like HTTP and HTTPS which are difficult to block without disrupting legitimate network traffic.

6- Persistence mechanisms are a hallmark of sophisticated ransomware. Identifying how persistence was achieved can aid in recovery and prevention of reinfection.

What is the MITRE ATT&CK Sub-Technique ID associated with the ransomware’s persistence technique? T1053.005

observing processes in any.run we can see the use of schtasks utility executed via cmd.exe to create a scheduled task with system privileges and run on the start with executing file like dispci.exe

7- As part of its infection chain, the ransomware created specific tasks to ensure its continued operation. Recognizing these tasks is crucial for system restoration. What are the names of the tasks created by the ransomware during execution?

rhaegal, drogon

we can see that it created two scheduled tasks, named rhaegal and drogon

rhaegal was designed to execute  dispci.exe

task drogon was created to initiate a forced system shutdown

8- the malicious binary dispci.exe displayed a suspicious message upon execution, urging users to disable their defenses. This tactic aimed to evade detection and enable the ransomware's full execution. What suspicious message was displayed in the Console upon executing this binary?

Disable your anti-virus and anti-malware programs

in this cisco talos report we see that the malware displed this message Disable your anti-virus and anti-malware programs

This message underscores the ransomware's intent to evade detection and mitigation by encouraging or coercing users to disable their security measures, thereby allowing the malicious payload to execute unimpeded. https://blog.talosintelligence.com/bad-rabbit/

To modify the Master Boot Record (MBR) and encrypt the victim’s hard drive, the ransomware utilized a specific driver. Recognizing this driver is essential for understanding the encryption mechanism.

What is the name of the driver used to encrypt the hard drive and modify the MBR? DiskCryptor

https://securelist.com/bad-rabbit-ransomware/82851/

DiskCryptor is a legitimate open-source full-disk encryption tool that has been co-opted by various ransomware families for malicious purposes. In this case, the ransomware leveraged DiskCryptor to execute its encryption mechanisms, effectively rendering the system inoperable without the decryption key.

9- Attribution is key to understanding the threat landscape. The ransomware was tied to a known attack group through its tactics, techniques, and procedures (TTPs).

What is the name of the threat actor responsible for this ransomware campaign? Sandworm

using Malpedia we can identify that the ransomware is associated with Sandworm

Sandworm is a notorious APT group known for its focus on cyber sabotage, particularly against critical infrastructure.

10- The ransomware rendered the system unbootable by corrupting critical system components. Identifying the technique used provides insight into its destructive capabilities.

What is the MITRE ATT&CK ID for the technique used to corrupt the system firmware and prevent booting? T1495

looking at the mitre techniques associated with the ransomware we find  T1495 Firmware Corruption

This form of sabotage targets essential parts of a system, such as the Master Boot Record (MBR) or firmware like the BIOS, which reside in non-volatile memory. By tampering with or destroying these components, attackers aim to lock users out of their machines entirely. The goal is to maximize disruption, making recovery difficult and thereby pressuring the victim into paying a ransom.
