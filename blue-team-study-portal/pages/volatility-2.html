Original Notes

Volatility 2

Setup

  • Specify Profile:

``bash vol.py --profile=Win7SP1x86 -f /path/to/memory.dmp ``

    • Choose the correct profile for the OS version.

OS Information

  • Image Info:

``bash vol.py -f /path/to/file imageinfo ``

    • Detects image profile and basic information.

Process Information

  • List Processes:

``bash vol.py -f /path/to/file pslist ``

    • Displays running processes.
  • Hidden Processes:

``bash vol.py -f /path/to/file psscan ``

    • Finds hidden/unlinked processes.
  • Process Tree:

``bash vol.py -f /path/to/file pstree ``

    • Shows parent-child process relationships.

Memory Dumping

  • Dump Process Executable:

``bash vol.py -f /path/to/file procdump -p <PID> -D /path/to/dir ``

    • Dumps an executable file from a specific process.

Handles

  • List Open Handles:

``bash vol.py -f /path/to/file handles -p <PID> ``

    • Displays handles (files, registry, etc.) open by a process.

DLLs

  • List DLLs:

``bash vol.py -f /path/to/file dlllist -p <PID> ``

    • Shows DLLs loaded by a process.

Command Line

  • Process Command Line:

``bash vol.py -f /path/to/file cmdline -p <PID> ``

    • Displays command-line arguments used for a process.

Network Information

  • Connections and Ports:

``bash vol.py -f /path/to/file netscan ``

    • Identifies active network connections and listening ports.

Registry Analysis

  • Registry Hives:

``bash vol.py -f /path/to/file hivelist #download ``

    • Lists registry hives.
  • Retrieve Registry Key:

``bash vol.py -f /path/to/file printkey -K "Software\Microsoft\Windows\CurrentVersion" ``

    • Displays values for a specific registry key.

File Analysis

  • File Scan:

``bash vol.py -f /path/to/file filescan ``

    • Scans for open file handles.
  • Dump File:

``bash vol.py -f /path/to/file dumpfiles -Q <OFFSET> -D /path/to/dir ``

    • Dumps files from memory.

Malware Analysis

  • Detect Code Injection:

``bash vol.py -f /path/to/file malfind -p <PID> ``

    • Scans for suspicious injected code.

Miscellaneous

  • Yara Scan:

``bash vol.py -f /path/to/file yarascan -y "rule { strings: $a = 'malicious' condition: $a }" ``

    • Performs YARA rule matching for malware detection.
  • Command Line Arguments (cmdline):

``bash vol.py -f /path/to/memory_dump --profile=<profile> cmdline ``

    • Extracts command-line arguments for processes running in memory. Useful for identifying executed files and their parameters.
  • Console Scan (consoles):

``bash vol.py -f /path/to/memory_dump --profile=<profile> consoles ``

    • Displays the content of console buffers, including command history and context of open terminals.
  • Network Connections Scan (connscan):

``bash vol.py -f /path/to/memory_dump --profile=<profile> connscan ``

    • Scans for TCP network connections. Helpful for identifying malicious connections initiated by executed files.
  • Shim Cache Scan (shimcache):

``bash vol.py -f /path/to/memory_dump --profile=<profile> shimcache ``

    • Lists recently executed files based on the Application Compatibility Cache, including their paths.
  • Handles Scan (handles):

``bash vol.py -f /path/to/memory_dump --profile=<profile> handles ``

    • Identifies open handles, including file handles, which might reveal file paths or objects used by a process.

Hashdump

bash
vol.py -f /path/to/file hashdump
  • Extracts user account password hashes from a Windows memory image.

Shutdowntime

bash
vol.py -f /path/to/file shutdowntime
  • Retrieves the last system shutdown time from the memory image.

Truecrypt Passphrase:

bash
vol.py -f /path/to/file truecryptpassphrase
  • Attempts to recover TrueCrypt passphrases from memory.

SSDT Plugin:

bash
vol.py -f /path/to/file ssdt
  • Lists the System Service Descriptor Table (SSDT) hooks, which can indicate rootkits or malicious modifications in a Windows kernel.

UserAssist (userassist):

bash
vol.py -f /path/to/memory_dump --profile=<profile> userassist
  • Extracts UserAssist registry keys from memory, which track user activity, such as applications executed.

Shellbags (shellbags):

bash
vol.py -f /path/to/memory_dump --profile=<profile> shellbags
  • Retrieves Shellbag artifacts from memory, providing information on directories accessed and file browsing history.

Environment Variables (envars):

bash
vol.py -f /path/to/memory_dump --profile=<profile> envars
  • Lists environment variables set in the system, potentially revealing useful information about system configuration or malware execution paths.

DLL Loading Analysis

  • List Loaded Modules with Discrepancies:

``bash vol.py -f /path/to/file ldrmodules -p <PID> ``

    • Lists loaded DLLs for a process and compares three lists: LoadOrder, InitOrder, and MemOrder.
    • Highlights discrepancies where a DLL is present in memory but not in all lists, which may indicate stealthy code injection or manually mapped DLLs.
    • Useful for detecting hidden modules used by malware or rootkits.
