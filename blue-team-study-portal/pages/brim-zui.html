Brim (zui)

Data Exploration

List unique log types (paths)

python
| cut _path | sort | uniq -c | sort -r

View sample events

python
select * from logs | head 10

Cutting & Filtering Fields

PurposeQuery

Basic field selection & dedup

python
_path == "conn"              # Connection logs
| cut id.orig_h, id.resp_h      # Show source & dest IP
| sort                          # Sort values alphabetically
| uniq -c | sort -r             # Count & rank unique pairs

Unique Connection & Frequency Queries

Unique network connections

python
_path == "conn"
| cut id.orig_h, id.resp_p, id.resp_h
| sort
| uniq

Unique DNS query counts

python
_path == "dns"
| count() by query
| sort -r

HTTP methods & URIs

python
_path == "http"
| cut id.orig_h, id.resp_h, id.resp_p, method, host, uri
| uniq -c | sort -r

Byte & Duration Metrics

Connection received data

python
_path == "conn"
| put total_bytes := orig_bytes + resp_bytes
| sort -r total_bytes
| cut uid, id.orig_h, id.resp_h, orig_bytes, resp_bytes, total_bytes

Top durations per connection

python
_path == "conn"
| cut id.orig_h, id.resp_p, id.resp_h, duration
| sort -r duration
| head 20

Suricata Alert Overview

Alerts by severity & category

python
event_type == "alert"
| count() by alert.severity, alert.category
| sort -r count

Suricata alerts by dest subnet

python
event_type == "alert"
| put dest_net := network_of(dest_ip)
| alerts := union(alert.category) by dest_net

Alerts with MITRE info

python
event_type == "alert"
| cut alert.category,
      alert.metadata.mitre_technique_name,
      alert.metadata.mitre_technique_id,
      alert.metadata.mitre_tactic_name
| uniq -c | sort -r
