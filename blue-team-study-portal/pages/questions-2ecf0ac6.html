1 - Open CMD and type “net user Administrator”. Which privileged group does the user belong to?

Why It Matters In SOC

Use this note as an analyst reference: understand the concept, identify the relevant telemetry, pivot through evidence, and document the decision path clearly.

Key Concepts

  • Questions
  • 1 - Open CMD and type “net user Administrator”. Which privileged group does the user belong to?
  • Administrators
  • 2 - Open Event Viewer and try to find your command in Sysmon logs. What is the “Image” field of the net command you just run?
  • C:\Windows\System32\net.exe
  • 3 - Looking at Sysmon logs, what is the first command the invoice.pdf.exe executes?
  • whoami
  • 4 - Which command did the malware use to check the presence of MS Defender EDR?
  • cmd /c “tasklist /v | findstr MsSense.exe || echo No MS Defender EDR”
  • 5 - To which domain did the malware send the discovered data?
  • exfil.beecz.cafe
  • 6 - What is the Facebook password that the user saved in Chrome? (Chrome menu Passwords and autofill Password Manager)

Analyst Workflow

  • Identify the user, host, process, network flow, or identity object involved.
  • Collect the highest-value logs first and preserve timestamps and assumptions.
  • Pivot through related notes, telemetry sources, and MITRE references.
  • Document findings, gaps, containment options, and detection improvements.

Questions

1 - Open CMD and type “net user Administrator”. Which privileged group does the user belong to?

Administrators

2 - Open Event Viewer and try to find your command in Sysmon logs. What is the “Image” field of the net command you just run?

C:\Windows\System32\net.exe

3 - Looking at Sysmon logs, what is the first command the invoice.pdf.exe executes?

whoami

4 - Which command did the malware use to check the presence of MS Defender EDR?

cmd /c “tasklist /v | findstr MsSense.exe || echo No MS Defender EDR”

5 - To which domain did the malware send the discovered data?

exfil.beecz.cafe

6 - What is the Facebook password that the user saved in Chrome? (Chrome menu > Passwords and autofill > Password Manager)

nsAghv51BBav90!

7 - Which interesting SSH key does the user store on disk? (Start your search from C:\Users\Administrator)

thm-access-database.key

8 - What is the secret PDF file explaining TryHackMe’s internal network?

thm-network-diagram-2025.pdf

9 - Looking at Sysmon logs, what directory does the stealer create?

staging_58f1

10 - Which three file extensions does the malware search for?

docx, pdf, xlsx

11 - Which PowerShell cmdlet does the malware use to get clipboard content?

Get-ClipBoard

12 - Which domain does the malware exfiltrate the data to?

collecteddata-storage-2025.s3.amazonaws.com

13 - Open the Chrome browser on the VM and navigate to the URL. What is the flag in the response?

THM{just_use_web_browser}

14 - Next, open CMD and download the file from the same URL using curl.exe. What is the flag in the response?

THM{curl_is_cool}

15 - Continue with the same CMD and URL, but now using certutil.exe. What is the flag in the response?

THM{abusing_certutil}

16 - Finally, download the same file using PowerShell IWR. What is the flag in the response?

THM{power_of_powershell}
