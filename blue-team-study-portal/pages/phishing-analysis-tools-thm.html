Phishing Case 1

Scenario: You are a Level 1 SOC Analyst. Several suspicious emails have been forwarded to you from other coworkers. You must obtain details from each email for your team to implement the appropriate rules to prevent colleagues from receiving additional spam/phishing emails.

1- What brand was this email tailored to impersonate?

Netflix

double click and it will automatically start thunderbird and you will see the attacker trying to imitate Netflix to get the user to click on the link embedded on that red button

2- What is the From email address?

NetfIixJGQ47wazXe1xYVBrkeDg-JOg7ODDQwWdR@JOg7ODDQwWdR-yVkCaBkTNp.gogolecloud.com*..***___

we don’t have to look to the source to answer this question can just click and select “Copy Email Address”

3- What is the originating IP? Defang the IP address.

209[.]85[.]167[.]226

for this we have to view the source

4- From what you can gather, what do you think will be a domain of interest? Defang the domain.

etekno[.]xyz

we can see the Return-Path header and the SPF header has domain not related to netflix at all

5- What is the shortened URL? Defang the URL.

hxxps[://]t[.]co/yuxfZm8KPg?amp==1

we can copy the link then head to CyberChef

Phishing Case 2

Scenario: You are a Level 1 SOC Analyst. Several suspicious emails have been forwarded to you from other coworkers. You must obtain details from each email for your team to implement the appropriate rules to prevent colleagues from receiving additional spam/phishing emails.

A malicious attachment from a phishing email inspected in the previous Phishing Room was uploaded to Any Run for analysis.

Task: Investigate the analysis and answer the questions below.

Link: https://app.any.run/tasks/8bfd4c58-ec0d-4371-bfeb-52a334b69f59

1- What does AnyRun classify this email as?

Suspicious activity

first thing you see after opening any.run report is the filename and classification that was marked by any.run

2- What is the name of the PDF file?

Payment-updateid.pdf

3- What is the SHA 256 hash for the PDF file?

cc6f1a04b10bcb168aeec8d870b97bd7c20fc161e8310b5bce1af8ed420e2c24

by clicking on the file name or opening the reports you will see the hashes if the file

4- What two IP addresses are classified as malicious? Defang the IP addresses. (answer: IP_ADDR,IP_ADDR)

2[.]16[.]107[.]24,2[.]16[.]107[.]83

will find the answer in the text report just copy it and defanged in CyberChef

5- What Windows process was flagged as Potentially Bad Traffic?

svchost.exe

back to the interactive report will find the flagged process in the “Threats” section

Phishing Case 3

Scenario: You are a Level 1 SOC Analyst. Several suspicious emails have been forwarded to you from other coworkers. You must obtain details from each email for your team to implement the appropriate rules to prevent colleagues from receiving additional spam/phishing emails.

A malicious attachment from a phishing email inspected in the previous Phishing Room was uploaded to Any Run for analysis.

Task: Investigate the analysis and answer the questions below.

Link: https://app.any.run/tasks/82d8adc9-38a0-4f0e-a160-48a5e09a6e83

1- What is this analysis classified as?

Malicious activity

will find the classification above the file name

2- What is the name of the Excel file?

CBJ200620039539.xlsx

3- What is the SHA 256 hash for the file?

5f94a66e0ce78d17afc2dd27fc17b44b3ffc13ac5f42d3ad6a5dcfb36715f3eb

click on the name of the file

4- What domains are listed as malicious? Defang the URLs & submit answers in alphabetical order. (answer: URL1,URL2,URL3)

biz9holdings[.]com,findresults[.]site,ww38[.]findresults[.]site

Connections section or from the text report

5- What IP addresses are listed as malicious? Defang the IP addresses & submit answers from lowest to highest. (answer: IP1,IP2,IP3)

75[.]2[.]11[.]242,103[.]224[.]182[.]251,204[.]11[.]56[.]48

the ips of the domains from the previous question

6- What vulnerability does this malicious attachment attempt to exploit?

CVE-2017-11882

when searching with it will find its a Buffer Overflow RCE on Microsoft Equation Editor of Microsoft Office which will be the initial access just like opening file with malicious macro but this have no macro just buffer overflow code

you can read more about this vulnerability in this articles

https://www.kaspersky.com/blog/cve-2017-11882-exploitation-on-the-rise/48768/

https://unit42.paloaltonetworks.com/unit42-analysis-of-cve-2017-11882-exploit-in-the-wild/
