PowerShell for Incident Response

Process Analysis

Purpose: Identify suspicious processes, malware, or abnormal execution paths.

Key Commands:

List running processes and basic info:

powershell
Get-Process | Select-Object Name, Id, CPU, Path | Sort-Object CPU -Descending

Check processes running from Temp or AppData (common malware locations):

powershell
Get-Process | Where-Object {$_.Path -like "*temp*" -or $_.Path -like "*appdata*"} | Select-Object Name,Id,Path

Build a process tree to detect suspicious parent-child chains:

powershell
Get-CimInstance Win32_Process | Select-Object ProcessId,ParentProcessId,CommandLine | Sort-Object ParentProcessId

Hunt for:

  • Office apps spawning PowerShell (winword.exe → powershell.exe)
  • Multiple PowerShell instances at unusual hours
  • Processes without valid vendor info or abnormal paths

PowerShell-Specific Investigation

Purpose: Detect malicious PowerShell activity, script execution, and injected code.

Key Commands:

Command lines for all PowerShell processes:

powershell
Get-CimInstance Win32_Process |
  Where-Object { $_.Name -match 'powershell|pwsh' -and $_.CommandLine -ne $null } |
  Select-Object ProcessId,CommandLine | Format-Table -Wrap

Check PowerShell binary hashes:

powershell
Get-Process -Name powershell*, pwsh* | ForEach-Object {
    [PSCustomObject]@{
        Name = $_.Name
        Path = $_.Path
        SHA256 = (Get-FileHash -Path $_.Path -Algorithm SHA256).Hash
    }
} | Format-Table -AutoSize

Extract script block execution logs (Event ID 4104):

powershell
Get-WinEvent -LogName "Microsoft-Windows-PowerShell/Operational" -MaxEvents 20 |
  Where-Object { $_.Id -eq 4104 } |
  Select-Object TimeCreated,@{N='Command';E={$_.Properties[0].Value}} | Format-Table -Wrap

Detect network connections initiated by PowerShell:

powershell
Get-NetTCPConnection -State Established |
  Where-Object { (Get-Process -Id $_.OwningProcess -ErrorAction SilentlyContinue).Name -match 'powershell|pwsh' } |
  Select-Object LocalAddress,RemoteAddress,RemotePort,@{N='Process';E={(Get-Process -Id $_.OwningProcess).Name}}

Hunt for PowerShell-created files in Temp and AppData:

powershell
Get-ChildItem -Path $env:TEMP,$env:APPDATA -Recurse -File |
  Where-Object { $_.CreationTime -gt (Get-Date).AddHours(-24) -and ($_.Name -like "*.ps1" -or "*.exe" -or "*.dll") } |
  Select-Object Name, FullName, CreationTime

Hunt for:

  • EncodedCommand / IEX patterns
  • AMSI bypass attempts
  • Unusual network connections to cloud providers or strange ports
  • PowerShell running from non-standard paths

Persistence and Scheduled Tasks

Purpose: Identify malware auto-start mechanisms.

Registry Auto-Start:

powershell
Get-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion\Run","HKLM:\Software\Microsoft\Windows\CurrentVersion\Run"

Scheduled Tasks:

powershell
Get-ScheduledTask | Where-Object { $_.State -ne "Disabled" } | Select-Object TaskName, Author

WMI Persistence:

Event Filters:

powershell
Get-WmiObject -Namespace root\Subscription -Class __EventFilter | Select-Object Name,Query

Event Consumers:

powershell
Get-WmiObject -Namespace root\Subscription -Class __EventConsumer | Select-Object Name,CommandLineTemplate

Filter-Consumer Bindings:

powershell
Get-WmiObject -Namespace root\Subscription -Class __FilterToConsumerBinding | Select-Object Filter,Consumer

Hunt for:

  • Tasks or WMI filters pointing to Temp/AppData files
  • Tasks created by PowerShell or unknown accounts

File and Execution History

Prefetch (program execution):

powershell
Get-ChildItem -Path "C:\Windows\Prefetch\*.pf" | Select-Object Name, LastAccessTime

LNK (recent shortcuts):

powershell
Get-ChildItem -Path "$env:APPDATA\Microsoft\Windows\Recent\*.lnk" | Select-Object Name, LastAccessTime

Recycle Bin:

powershell
Get-ChildItem -Path "C:\`$Recycle.Bin" -Recurse -Force | Select-Object FullName, LastWriteTime

ShimCache (AppCompatCache):

powershell
Get-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\Session Manager\AppCompatCache" -Name AppCompatCache

UserAssist History:

powershell
Get-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\UserAssist\*\Count"

Hunt for:

  • Executables recently run from unusual locations
  • Deleted malware artifacts
  • Evidence of lateral movement or data staging

Network & Connections

Active connections:

powershell
Get-NetTCPConnection -State Established | Select-Object LocalAddress,RemoteAddress,OwningProcess

Anomalous processes:

powershell
Get-NetTCPConnection -State Established | Where-Object {
    $_.RemoteAddress -notmatch '^(127\.|0\.0\.0|::1)' -and
    (Get-Process -Id $_.OwningProcess).Name -match 'svchost|powershell|wscript'
} | Select-Object LocalAddress, LocalPort, RemoteAddress, RemotePort, State,
    @{Name="Process"; Expression={(Get-Process -Id $_.OwningProcess).Name}},
    @{Name="PID"; Expression={$_.OwningProcess}},
    @{Name="ProcessPath"; Expression={(Get-Process -Id $_.OwningProcess).Path}}

DNS cache:

powershell
Get-DnsClientCache | Where-Object { $_.Entry -notmatch "microsoft|windows" }

Hunt for:

  • Remote connections to unknown IPs
  • Suspicious ports (443, 8443, 4443) without proper SSL
  • Connections initiated by unusual parent processes

Event Logs & Windows Artifacts

Authentication (4624):

powershell
Get-WinEvent -FilterHashtable @{LogName='Security'; ID=4624} -MaxEvents 20 | Select-Object TimeCreated,Message

PowerShell Operational Logs (4104):

powershell
Get-WinEvent -LogName "Microsoft-Windows-PowerShell/Operational" -MaxEvents 15 | Where-Object {$_.Id -eq 4104} | Select-Object Message

Defender Detection (1116):

powershell
Get-WinEvent -LogName "Microsoft-Windows-Windows Defender/Operational" | Where-Object {$_.Id -eq 1116} | Select-Object -First 10

File access tracking (4663):

powershell
Get-WinEvent -FilterHashtable @{LogName='Security';ID=4663} -MaxEvents 1000 | ? { $_.Properties[6].Value -like '*\malware.exe' } | select TimeCreated,@{N='Process';E={$_.Properties[6].Value}},@{N='File';E={$_.Properties[1].Value}}

Hunt for:

  • Unauthorized logins or lateral movement
  • Suspicious PowerShell command execution
  • Access to sensitive files by unusual processes

Forensics Checks

USB History:

powershell
Get-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Enum\USBSTOR\*\*" | Select-Object FriendlyName

BITS Transfers:

powershell
Get-BitsTransfer | Where-Object {$_.JobState -ne "Transferred"} | Select-Object DisplayName,FileList

Shellbags:

powershell
Get-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\Shell\Bags\*\Shell" | Select-Object PSChildName,ItemPos*

Group Policy Changes (last 7 days):

powershell
Get-ChildItem -Path "C:\Windows\System32\GroupPolicy\DataStore\*\*" -Recurse -Force | Where-Object {$_.LastWriteTime -gt (Get-Date).AddDays(-7)}

Volume Shadow Copies:

powershell
vssadmin list shadows

Environment Variables:

powershell
Get-ItemProperty -Path "HKLM:\System\CurrentControlSet\Control\Session Manager\Environment" | Select-Object Path,PSChildName,*
