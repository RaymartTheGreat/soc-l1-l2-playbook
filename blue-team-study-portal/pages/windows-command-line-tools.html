Original Notes

Windows command-line & tools

System & Processes

  1. whoami /all — show current user, groups and privileges.
  2. tasklist / tasklist /v /svc — list running processes, PIDs, services hosted, and verbose info.
  3. taskkill /IM <name> /PID <pid> /F — terminate a process by name or PID.
  4. wmic process get ProcessId,ParentProcessId,ExecutablePath,CommandLine — enumerate processes with paths and command lines.
  5. Get-CimInstance Win32_Process (PowerShell) — fetch process objects with command line and metadata.
  6. procdump -ma <PID> <outfile> (Sysinternals) — create a full memory dump of a process.
  7. handle.exe <process|file> (Sysinternals) — list open handles for processes and files.

Users, Groups & Accounts

  1. net user — list local user accounts or show details for a specific user.
  2. net user <username> /add / /delete — create or delete local user accounts.
  3. net localgroup — list local groups.
  4. net localgroup Administrators — list members of the Administrators group.
  5. Get-LocalUser (PowerShell) — list local user accounts.
  6. Get-LocalGroupMember Administrators (PowerShell) — list members of a local group.

Network & Name Resolution

  1. ipconfig /all — show IP addresses, MACs, DNS and adapter info.
  2. getmac — display MAC addresses for network adapters.
  3. netstat -ano — list active network connections, listening ports and PIDs.
  4. netstat -anob — same as above with owning executable names (requires admin).
  5. nslookup <name> — query DNS records.
  6. tracert <host> — trace network route to a destination.
  7. pathping <host> — combined traceroute and packet-loss analysis.
  8. arp -a — display ARP cache (IP-to-MAC mappings).
  9. route print — show IP routing table.
  10. netsh (subcommands) — configure network interfaces, firewall, and traces (e.g., netsh interface ip show config).
  11. netsh trace start/stop — capture a network trace to an ETL file.

Services, Scheduling & Persistence

  1. sc query / sc qc <ServiceName> — query service state and configuration.
  2. service control (net start/net stop) — start or stop services.
  3. schtasks /query /fo LIST /v — list scheduled tasks with verbose details.
  4. schtasks /create /tn "<name>" /tr "<command>" /sc <schedule> — create a scheduled task.
  5. at (legacy) — legacy task scheduler utility (mostly superseded by schtasks).
  6. reg query HKLM\...\Run /s — query Run/RunOnce registry persistence locations.
  7. sc create / sc delete — create or delete Windows services.

File System & Disk

  1. dir / dir /a /s /b — list files and directories (with hidden/system and recursion options).
  2. type / more — display file contents.
  3. copy / xcopy / robocopy — file copy utilities (robocopy for robust sync/mirroring).
  4. del / erase — delete files.
  5. move / ren — move or rename files/directories.
  6. attrib — view/change file attributes (hidden, system, read-only).
  7. takeown /f <path> — take ownership of a file or directory.
  8. icacls <path> — view or modify NTFS permissions.
  9. fsutil — advanced filesystem operations (USN journal, ADS, volume info).
  10. cipher — manage EFS and wipe free space.
  11. compact — view/modify NTFS compression.
  12. diskpart — disk and partition management (interactive).
  13. format — format volumes.
  14. chkdsk — check and optionally repair disk errors.

Event Logs & System Info

  1. wevtutil — manage and query Windows event logs (export, clear, query).
  2. eventvwr — open Event Viewer GUI.
  3. systeminfo — display OS, hardware and patch (hotfix) info.
  4. wmic qfe list brief — list installed hotfixes/patches.
  5. get-computerinfo (PowerShell) — detailed system information.
  6. sfc /scannow — system file checker to verify/repair Windows protected files.

PowerShell (core cmdlets & commonly abused)

  1. Get-Process — list running processes.
  2. Stop-Process -Id <PID> — terminate a process.
  3. Get-Service — list Windows services.
  4. Start-Service / Stop-Service — start or stop a service.
  5. Get-ScheduledTask — list scheduled tasks.
  6. Register-ScheduledTask / Unregister-ScheduledTask — create or remove scheduled tasks.
  7. Get-NetIPConfiguration / Get-NetIPAddress — show network configuration.
  8. Get-NetTCPConnection — list TCP connections (PowerShell view of netstat).
  9. Get-LocalUser / Get-LocalGroup / Get-LocalGroupMember — enumerate local users/groups and members.
  10. Get-CimInstance -ClassName Win32_Process — WMI/CIM process enumeration (incl. command line).
  11. Get-WmiObject / Get-CimInstance -ClassName <class> — legacy/alternate WMI queries.
  12. Get-WinEvent -LogName Security — query Windows event logs.
  13. Get-FileHash <path> -Algorithm SHA256 — compute file hash.
  14. Get-Content <path> — read file contents (like type).
  15. Get-ChildItem -Recurse -Force — list files and folders recursively (like dir /s).
  16. Set-Content / Add-Content — write/append file content.
  17. Remove-Item <path> -Recurse -Force — delete files/folders/registry keys.
  18. Invoke-Command -ComputerName <host> -ScriptBlock {} — execute commands remotely (PowerShell Remoting).
  19. New-PSSession / Enter-PSSession / Get-PSSession — manage interactive/remote sessions.
  20. Invoke-Expression (IEX) — execute a string as code (commonly abused for one-liners).
  21. Invoke-WebRequest -Uri <url> -OutFile <file> — download files/web content.
  22. Invoke-RestMethod -Uri <url> — interact with web APIs (and C2-style callbacks).
  23. (New-Object System.Net.WebClient).DownloadFile('<url>','<path>') — alternate file download method.
  24. Start-Process -FilePath <path> -WindowStyle Hidden -NoNewWindow — launch a process (options for stealth).
  25. Start-Job / Get-Job / Receive-Job — run background jobs.
  26. Get-Content $env:windir\system32\drivers\etc\hosts — read hosts file (local DNS overrides).
  27. Get-ItemProperty -Path HKLM:\Software\... — read registry values.
  28. Set-ItemProperty -Path HKLM:\Software\... -Name <value> -Value <data> — write registry values.
  29. New-PSDrive -Name X -PSProvider FileSystem -Root \\host\share — map a network share as a drive.
  30. Get-Credential — create a credential object (used for authentication in scripts).
  31. ConvertTo-SecureString / ConvertFrom-SecureString — encrypt/decrypt strings for credential handling.
  32. Get-ADUser / Get-ADComputer / Get-ADGroupMember / Search-ADAccount (ActiveDirectory module) — AD enumeration and discovery.
  33. Resolve-DnsName <host> — DNS resolution (PowerShell replacement for nslookup).
  34. Get-ExecutionPolicy / Set-ExecutionPolicy — view/change PowerShell execution policy.
  35. Add-Type -AssemblyName <assembly> — load/compile .NET code into the session.
  36. Get-MpComputerStatus — query Windows Defender status (defender module).
  37. Export-Csv / Import-Csv — export/import tabular data to/from CSV.
  38. Export-Clixml / Import-Clixml — export/import structured objects with metadata.
  39. Out-File / Tee-Object — save or tee command output to files.
  40. Start-Process -Credential <pscredential> -FilePath <exe> — start process as a different user (when supported).

Backup, Shadow Copies & Recovery

  1. vssadmin list shadows — list Volume Shadow Copies.
  2. vssadmin delete shadows — delete shadow copies (destructive).
  3. wbadmin — backup and recovery management (Windows Backup).

Certificates, Downloads & Transfer

  1. certutil -hashfile <file> SHA256 — compute file hash; also supports encoding and downloads (urlcache).
  2. certutil -urlcache -split -f <URL> <outfile> — download file using certutil.
  3. bitsadmin /list /allusers — list BITS jobs (legacy); create/query background transfers.
  4. bitsadmin /reset — cancel BITS jobs.

Audit, Policy & Security Configuration

  1. auditpol /get /category:* — view audit policies.
  2. gpresult /r — show applied Group Policy settings.
  3. gpupdate /force — refresh Group Policy.
  4. secedit — configure security templates and policies.
  5. whoami /priv — list user privileges.

Misc & Helpers

  1. clip — pipe output to Windows clipboard (e.g., ipconfig /all | clip).
  2. clip < file — copy file contents to clipboard.
  3. shutdown /s /r /a — shutdown, restart, or abort scheduled shutdown.
  4. mmc / mmc.exe — open Microsoft Management Console.
  5. msinfo32 — open System Information GUI or save inventory.
  6. resmon / perfmon / taskmgr — Resource Monitor, Performance Monitor, Task Manager GUIs.
  7. winget — Windows package manager (list/install apps).

Sysinternals (recommended tools)

  1. Process Explorer (procexp.exe) — advanced process and handle/DLL inspection.
  2. Process Monitor (procmon.exe) — realtime file/registry/process/network API monitor.
  3. Autoruns / autorunsc — enumerate all autorun/startup locations.
  4. PsExec (psexec.exe) — execute processes remotely.
  5. Sigcheck (sigcheck.exe) — check file signatures and VirusTotal integration.
  6. TCPView (tcpview.exe) — realtime view of network endpoints per process.
  7. PsList / PsService / PsLoggedOn — assorted PsTools for processes, services, and logged-on users.
  8. BgInfo — display system information on desktop wallpaper (useful for quick host context).
