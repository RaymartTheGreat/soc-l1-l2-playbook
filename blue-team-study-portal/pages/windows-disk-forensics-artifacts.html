Windows Disk Forensics Artifacts

🔍 Finding Malicious Executables & Process IDs (PIDs)

  • Automated Timeline:

Use Hayabusa to generate a timeline from EVTX logs:

``python hayabusa.exe json-timeline -d <evtx_directory> -o output.json ``

  • Manual Artifact Review:
    • Cross-reference EVTX logs manually.
    • Check recently opened LNK files at:

``markup C:\Users\<User>\AppData\Roaming\Microsoft\Windows\Recent ``

🐀 Persistence Mechanisms

🗂 Registry Keys

  • Startup Programs:
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
    • HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\RunOnce
    • HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\RunOnce
  • Services & Drivers:
    • HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services
  • Scheduled Tasks:
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Schedule\TaskCache\Tasks
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Schedule\TaskCache\Tree
  • Winlogon Shell/Userinit Persistence:
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon

📁 File System

  • Scheduled Tasks:
    • C:\Windows\System32\Tasks

🌐 Browser Extensions

  • Chrome:

C:\Users\<User>\AppData\Local\Google\Chrome\User Data\Default\Extensions

  • Firefox:

C:\Users\<User>\AppData\Roaming\Mozilla\Firefox\Profiles\<Profile>\extensions

🖥 System Information

Used for profiling the host system.

  • Windows Version and Install Date:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion

  • Computer Name:

HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\ComputerName\ComputerName

  • Timezone Information:

HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\TimeZoneInformation

🌐 Network Information

  • Physical Network Cards:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\NetworkCards

  • TCP/IP Interface Config:

HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Services\Tcpip\Parameters\Interfaces

  • Network Connection History:
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\NetworkList\Signatures\Unmanaged
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\NetworkList\Profiles
    • Microsoft-Windows-WLAN-AutoConfig/Operational.evtx

👤 User Account Activity

  • User Metadata (Name, SID, Login Time):

Found in the SAM hive (use tools like regripper)

  • Login/Logout and User Events (from Security.evtx):
    • 4624 – Successful logon
    • 4625 – Failed logon
    • 4634 – Logoff
    • 4647 – User initiated logoff
    • 4672 – Special privilege logon
    • 4648 – Logon with alternate credentials
    • 4720 / 4726 – Account creation / deletion

📁 File Activities

  • NTFS File System Traces:
    • $MFT, $LogFile, $UsnJrnl contain filename, timestamps, and rename activity.
    • Tool: NTFS Log Tracker
  • Deleted File Evidence:
    • $I30 records in directories (indexed filenames).
    • Tool:

``bash INDXRipper.exe <image_path> <output_file> ``

  • Audit Logs for Access Attempts:
    • Logged in Security.evtx (if object access auditing is enabled via GPO).

⚙️ Execution Artifacts

  • Prefetch Files:

C:\Windows\Prefetch

  • Shimcache (AppCompatCache):

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\AppCompatCache

  • Amcache:

C:\Windows\AppCompat\Programs\Amcache.hve

  • Recent Files (LNK Shortcuts):

C:\Users\<User>\AppData\Roaming\Microsoft\Windows\Recent

  • Jump Lists:

C:\Users\<User>\AppData\Roaming\Microsoft\Windows\Recent\AutomaticDestinations

📌 High-Value Artifacts

  • LNK Files – shortcut traces from user interactions
  • Recycle Bin Contents:

C:\$Recycle.Bin

  • Browser History:
    • Chrome: C:\Users\<User>\AppData\Local\Google\Chrome\User Data\Default\History
    • Firefox: C:\Users\<User>\AppData\Roaming\Mozilla\Firefox\Profiles\<Profile>\places.sqlite
  • Mounted Volumes:

HKEY_LOCAL_MACHINE\SYSTEM\MountedDevices

  • USB Storage Devices:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\USBSTOR

🔌 Connected Devices

  • USB Enumeration:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\USB

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\USBSTOR

  • UserAssist (GUI Program Launch History):

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\UserAssist
