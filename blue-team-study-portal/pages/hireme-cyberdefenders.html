Scenario

Karen is a security professional looking for a new job. A company called "TAAUSAI" offered her a position and asked her to complete a couple of tasks to prove her technical competency. As a soc analyst Analyze the provided disk image and answer the questions based on your understanding of the cases she was assigned to investigate.

https://cyberdefenders.org/blueteam-ctf-challenges/hireme/

First we open ftk imager and export important hives then open it using registry explorer

1- What is the administrator's username?

Karen

we can see it in registry explorer but to make sure we will use regripper on sam hive

then we search with the SID we found it in the Administrators group

2- What is the OS's build number?

16299

Microsoft\Windows NT\CurrentVersion

3- What is the hostname of the computer?

TOTALLYNOTAHACK

ControlSet001\Control\ComputerName\ComputerName

4- A messaging application was used to communicate with a fellow Alpaca enthusiest. What is the name of the software?

skype

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\App Paths

5- What is the zip code of the administrator's post?

19709

To get this information we have to check the file which stores Browser data and history .

root\Users\Karen\AppData\Local\Google\Chrome\User Data\Default\Web Data

6- What are the initials of the person who contacted the admin user from TAAUSAI?

MS

for this we have to check outlook artifacts and we can’t open it using ftk imager so we export it and open it with ost viewer

7- How much money was TAAUSAI willing to pay upfront?

150000

check more emails

8- What country is the admin user meeting the hacker group in?

Egypt

and more

9- What is the machine's timezone? (Use the three-letter abbreviation)

UTC

System\ControlSet001\Control\TimeZoneInformation

10- When was AlpacaCare.docx last accessed?

03/17/2019 09:52 PM

go to the docx file in ftx imager then look at file properties

11-There was a second partition on the drive. What is the letter assigned to it?

A

SYSTEM\MountedDevices

12- What is the answer to the question Company's manager asked Karen?

TheCardCriesNoMore

we see the manger give a task to decode this base64 we decode it and got the answer

also if we look at the replays we will get the answer

13- What is the job position offered to Karen? (3 words, 2 spaces in between)

Cyber Security Analyst

check the next email

14- When was the admin user password last changed?

03/21/2019 19:13:09

from the output of regripper

15- What version of Chrome is installed on the machine?

72.0.3626.121

SOFTWARE\WOW6432Node\Microsoft\Windows \CurrentVersion\Uninstall\Google Chrome

16- What is the URL used to download Skype?

https://download.skype.com/s4l/download/win/Skype-8.41.0.54.exe

tried looking in software hive using registry explorer but didn’t find it so I examined the $MFT (Master File Table) for evidence of file downloads

17- What is the domain name of the website Karen browsed on Alpaca care that the file AlpacaCare.docx is based on?

palominoalpacafarm.com

export AlpacaCare.docx and analyze it
