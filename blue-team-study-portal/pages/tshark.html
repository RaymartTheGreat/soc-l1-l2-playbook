Capture Autostop & Ring Buffers

One-shot stop conditions (-a)

bash
# Stop after 30s
tshark -i eth0 -w file.pcap -a duration:30
# Stop at 10 MB
tshark -i eth0 -w file.pcap -a filesize:10240
# Rotate into 3 files of ≤10 MB each
tshark -i eth0 -w file.pcap -a filesize:10240 -a files:3

Continuous ring buffer (-b)

bash
# New file every 60s, keep last 5
tshark -i eth0 -w rotating.pcap -b duration:60 -b files:5
# Rotate by filesize
tshark -i eth0 -w rotating.pcap -b filesize:5120 -b files:4
  • Autostop (a) stops the capture once a single condition (e.g. duration or max size) is met.
  • Ring buffer (b) continuously rotates through multiple files, overwriting the oldest when a threshold is reached.

Capture vs. Display Filters

Filter TypeOptionSyntaxDescription
Capture (BPF)-f"host 10.0.0.5"Filters at packet capture time
"net 192.168.1.0/24"
"port 80"
Display-Y'ip.addr == 10.0.0.5'Filters after packets are captured/read
'tcp.port == 443'
'http.request.method == "GET"'

BPF Qualifiers

QualifierMeaningExamples
hostSingle IP or hostname-f "host 1.2.3.4"
netIP subnet-f "net 10.0.0.0/16"
portSingle port-f "port 53"
portrangePort range-f "portrange 1000-2000"
src/dstSource or destination only-f "src host 5.6.7.8"
protocolBy protocol name or number-f "tcp" / -f "ip proto 1" (ICMP)

Display Filter Examples

bash
# IP filtering
tshark -Y 'ip.src == 10.10.10.10'

# TCP ports
tshark -Y 'tcp.dstport == 22'

# HTTP traffic & status
tshark -Y "http && http.response.code == 200"

# DNS queries
tshark -Y 'dns.qry.type == 1'

# Combine filters
tshark -Y 'ip.addr == 192.168.1.5 && tcp.port == 443'

Statistics & Analysis (z)

bash
# Quick protocol hierarchy
tshark -z io,phs

# HTTP request statistics
tshark -z http,tree -q

# Top talkers by IP
tshark -z endpoints,ip

# DNS query tree
tshark -z dns,tree -q

# Follow streams
# TCP flow
tshark -z follow,tcp,ascii,0

(Append -q to suppress raw packet list and show only stats.)

Exporting Objects

bash
# Extract HTTP, SMB, TFTP objects to folder
tshark -r capture.pcap -q --export-objects "http,./http_objs" --export-objects "smb,./smb_objs"

Field Extraction (T fields)

bash
# Print src and dst IP with headers
tshark -r capture.pcap -T fields -e ip.src -e ip.dst -E header=y

# Extract DHCP hostnames
tshark -r dhcp.pcap -T fields -e dhcp.option.hostname | awk NF | sort | uniq -c | sort -r

# Count unique User-Agents
tshark -r web.pcap -T fields -e http.user_agent | sort | uniq -c | sort -r

Use awk, sort, and uniq to refine and summarize output.
