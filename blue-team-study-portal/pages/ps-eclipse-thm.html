PS Eclipse (THM)

Scenario

You are a SOC Analyst for an MSSP (Managed Security Service Provider) company called TryNotHackMe .

A customer sent an email asking for an analyst to investigate the events that occurred on Keegan's machine on Monday, May 16th, 2022 . The client noted that the machine is operational, but some files have a weird file extension. The client is worried that there was a ransomware attempt on Keegan's device.

Your manager has tasked you to check the events in Splunk to determine what occurred in Keegan's device.

https://tryhackme.com/room/posheclipse

1- A suspicious binary was downloaded to the endpoint. What was the name of the binary?

OUTSTANDING_GUTTER.exe

filter with event code 1 with table and dedup to command line looking for any curl, mshta , certuil , powershelland we got this two suspicious powershell with base64 and after each of them is a schedule task creation to execute C:\Windows\Temp\COUTSTANDING_GUTTER.exe as system after decoding it using cyberchef it we see that it disable windows defender real time monitoring and uses wget to download OUTSTANDING_GUTTER.exe from ngrok url to Temp directory then it creates schedule task

2- What is the address the binary was downloaded from? Add http:// to your answer & defang the URL.

hxxp[://]886e-181-215-214-32[.]ngrok[.]io

already found the address from cyberchef we just defang it

3- What Windows executable was used to download the suspicious binary? Enter full path.

C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe

already know from Q1 it was downloaded by powershell just copy the path from the log

4- What command was executed to configure the suspicious binary to run with elevated privileges?

"C:\Windows\system32\schtasks.exe" /Create /TN OUTSTANDING_GUTTER.exe /TR C:\Windows\Temp\COUTSTANDING_GUTTER.exe /SC ONEVENT /EC Application /MO *[System/EventID=777] /RU SYSTEM /f

from Q1 we see the commands that was executed after the powershell encoded command and from the decoded powershell we know it run as System

5- What permissions will the suspicious binary run as? What was the command to run the binary with elevated privileges? (Format: User + ; + CommandLine)

NT AUTHORITY\SYSTEM;"C:\Windows\system32\schtasks.exe" /Run /TN OUTSTANDING_GUTTER.exe

6- The suspicious binary connected to a remote server. What address did it connect to? Add http:// to your answer & defang the URL.

hxxp[://]9030-181-215-214-32[.]ngrok[.]io

we filter with event code 22 (DNS query detected) and defanged with cyberchef

7- A PowerShell script was downloaded to the same location as the suspicious binary. What was the name of the file?

script.ps1

so i filter with \\Temp\\.ps1* to see any scripts in the Temp directory maybe i should add Event code 11

another way to solve it is with the process id

8- The malicious script was flagged as malicious. What do you think was the actual name of the malicious script?

BlackSun.ps1

search the script hash on virustotal and we can see that this powershell script is the BlackSun ransomware

9- A ransomware note was saved to disk, which can serve as an IOC. What is the full path to which the ransom note was saved?

C:\Users\keegan\Downloads\vasg6b0wmw029hd\BlackSun_README.txt

now that we know its blacksun ransomware i found this blog which helps in solving this last 2 questions it have the name of the ransom note and the name of the image that replace the victim desktop

https://blogs.vmware.com/security/2022/01/blacksun-ransomware-the-dark-side-of-powershell.html

and if we didn’t read any blogs we can just search with event code 11 and *.txt and we could powershell as well

10- The script saved an image file to disk to replace the user's desktop wallpaper, which can also serve as an IOC. What is the full path of the image?

C:\Users\Public\Pictures\blacksun.jpg

just search with blacksun.jpg as in the blog
