so will check Thumbnail Caches

  • Thumbnail Caches

store miniature preview images of files (like pictures, videos, or documents) that Windows generates for faster display in Explorer.

    • Located in C:\Users\<User>\AppData\Local\Microsoft\Windows\Explorer\thumbcache_*.db.
    • Even if the original file is deleted, its thumbnail may still remain in the cache.
    • Useful for investigators to identify viewed images or files, confirm file presence, and recover partial evidence.

Windows Explorer generates thumbnail previews when a user browses image or video files. These previews are stored in .db files within the user’s profile at:

C:\Users\<USERNAME>\AppData\Local\Microsoft\Windows\Explorer\

The file thumbcache_256.db in particular stores 256×256 pixel thumbnails, which are often retained even after the source file is removed or overwritten.

open it with Thumbcache Viewer and locate the png click on it and we have the time
