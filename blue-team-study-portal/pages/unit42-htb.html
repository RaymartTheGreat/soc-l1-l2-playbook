Unit42 (HTB)

Scenario

Palo Alto's Unit42 recently conducted research on an UltraVNC campaign, wherein attackers utilized a backdoored version of UltraVNC to maintain access to systems. This lab is inspired by that campaign and guides participants through the initial access stage of the campaign.

https://app.hackthebox.com/sherlocks/Unit42

1- How many Event logs are there with Event ID 11?

56

no explanation needed just filter with event code 11 (file creation) using filter current log

2- Whenever a process is created in memory, an event with Event ID 1 is recorded with details such as command line, hashes, process path, parent process path, etc. This information is very useful for an analyst because it allows us to see all programs executed on a system, which means we can spot any malicious processes being executed. What is the malicious process that infected the victim's system?

C:\Users\CyberJunkie\Downloads\Preventivo24.02.14.exe.exe

we can see a suspicious executable with 2 .exe extension and was executed from the downloads directory so we search its hash on VirusTotal we confirm that this executable is malicious and its reported as UltraVNC RAT

3- Which Cloud drive was used to distribute the malware?

dropbox

Cloud services are a common method used by attackers to distribute malware because they aren’t flagged as malicious

we will look for event code 22 which detect any dns query made . we find the use of Dropbox a cloud file storage site and right after it we see event code 11 file creation of Prevntivo24.02.14.exe.exe with Firefox based on the timing of download of the file we sure that dropbox was used to deliver the malware

4- For many of the files it wrote to disk, the initial malicious file used a defense evasion technique called Time Stomping, where the file creation date is changed to make it appear older and blend in with other files. What was the timestamp changed to for the PDF file?

2024-01-14 08:10:06

filter with event code 2 which records any change of file creation time

5- The malicious file dropped a few files on disk. Where was "once.cmd" created on disk? Please answer with the full path along with the filename.

C:\Users\CyberJunkie\AppData\Roaming\Photo and Fax Vn\Photo and vn 1.1.2\install\F97891C\WindowsVolume\Games\once.cmd

filter with event code 11 (file creation) and search for once.cmd we get 2 results one where file was created by msiexec and another where it was created by preventivo24.02.14.exe

6- The malicious file attempted to reach a dummy domain, most likely to check the internet connection status. What domain name did it try to connect to?

www.example.com

again with event code 22 we see the malicious executable attempts to communicate with example.com domain

7- Which IP address did the malicious process try to reach out to?

93.184.216.34

filter with event code 3 (network connection) / we see the malicious process communicating with 93.184.216.34

8- The malicious process terminated itself after infecting the PC with a backdoored variant of UltraVNC. When did the process terminate itself?

2024-02-14 03:41:58

filter with event code 5 (process termination)
