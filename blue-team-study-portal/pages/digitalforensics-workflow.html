Original Notes

Digital‑Forensics Workflow

1- Chain‑of‑Custody & Documentation

  • Label evidence (drive tags, case numbers, date/time, collector).
  • Photograph device and port configuration.
  • Log every action (who, what, when, why).

2- Evidence Acquisition (Imaging)

  • Use a hardware write‑blocker or live‑response script to prevent writes.
  • Create a bit‑for‑bit image (E01, DD/raw, AFF).
  • Compute hashes (MD5, SHA‑1, SHA‑256).
  • Record imaging tool/version, exact command‑line, start/stop times.
  • Tools: Hardware write‑blocker, FTK Imager, dd, Magnet Acquire

3- Verification

  • Re‑calculate hashes on the mounted image to confirm integrity.
  • Compare to original hash—only proceed if they match.
  • Tools: FCIV, md5deep/sha256deep, HashCalc

4- Mounting the Image Read‑Only

  • Verify that the mount is read‑only before proceeding.
  • Tools (Windows): Arsenal Image Mounter, AVFS
  • Tools (Linux/macOS): ewfmount, mount -o loop,ro

5- Triage & Artifact Collection with KAPE

  • Tools: KAPE, Rekall/Volatility (for memory), Log2timeline/Plaso
  • Targets (raw files/folders to grab):
    • Event logs: Windows\System32\winevt\Logs\*.evtx
    • Registry hives: SYSTEM, SAM, SOFTWARE, NTUSER.DAT
    • USN Journal: \$Extend\$UsnJrnl\UsnJrnl
    • Prefetch: Windows\Prefetch\*.pf
    • LNK/Jump Lists: Users\\AppData\Roaming\Microsoft\Windows\Recent\
  • Modules (parsers): DefaultParser, ShimcacheParser, AmcacheParser
  • Sample command:

``powershell kape.exe --tsource M: --tdest C:\KAPE-Out \ --target BasicCollection,UsnJrnl,Prefetch \ --module DefaultParser,ShimcacheParser,AmcacheParser ``

6- “Important Stuff” Extraction & Initial Review

  • Tools: Autopsy, FTK Imager (Logical view), Excel/Splunk
  • Focus on:
    • User activity: shellbags, RecentDocs, LNK files, USB‑MRU
    • Logon/logoff: Event IDs 4624, 4634, Sysmon 1, 4688
    • Persistence: Run/RunOnce keys, Scheduled Tasks, Services, WMI filters
    • Code execution: Prefetch, AmCache, ShimCache entries
    • Malware indicators: unusual file paths, suspicious registry mods

7- Full Analysis & Timeline

  • Tools: Log2timeline/Plaso, Timesketch, Splunk
  • Consolidate CSV/JSON outputs into a timeline.
  • Correlate events across hosts/users.

8- Reporting & Remediation Recommendations

  • Document findings with timestamps, hashes, paths, Event IDs.
  • Provide IOC list and containment steps.
