Insider (CyberDefenders)

Scenario

After Karen started working for 'TAAUSAI,' she began doing illegal activities inside the company. 'TAAUSAI' hired you as a soc analyst to kick off an investigation on this case.

You acquired a disk image and found that Karen uses Linux OS on her machine. Analyze the disk image of Karen's computer and answer the provided questions.

https://cyberdefenders.org/blueteam-ctf-challenges/insider/

Q1 : Which Linux distribution is being used on this machine?

kali

The syslogfile in /var/log/installer/ contains detailed installation logs, including kernel initialization messages and boot parameters. Examining the file reveals a line indicating the kernel version: Linux version 4.13.0-kali1-amd64, confirming that the system is running Kali Linux

Q2 : What is the MD5 hash of the Apache access.log file?

d41d8cd98f00b204e9800998ecf8427e

Using FTK Imager, the MD5 hash of the file can be calculated, with the hash report highlighting the MD5 value.

The access.log file records all HTTP requests made to the Apache server, including client IPs, timestamps, requested URLs, HTTP methods, response codes, and user-agent information. This log is crucial for monitoring server activity and detecting potential security incidents.

Q3 : It is suspected that a credential dumping tool was downloaded. What is the name of the downloaded file?

mimikatz_trunk.zip

By examining the disk image in FTK Imager, the /root/Downloads/ directory was inspected, revealing a file named mimikatz_trunk.zip.

Mimikatzis a post-exploitation tool that can extract plaintext passwords, hashes, PINs, and Kerberos tickets from memory, perform pass-the-hash attacks, manipulate authentication tokens, and export certificates. While used by penetration testers for security assessments, it is also commonly exploited by attackers for credential theft.

Q4: A super-secret file was created. What is the absolute path to this file?

/root/Desktop/SuperSecretFile.txt

By examining the .bash_history file in the /root/ directory, we can track commands executed by the root user. Among these, a command was found that created a file named SuperSecretFile.txt on the /root/Desktop/ directory. The command uses touch to create the file, with output redirection (>) writing data into SuperSecretFile.txt, making /root/Desktop/SuperSecretFile.txt the absolute path to the super-secret file.

The .bash_history file, located in a user’s home directory, logs all commands executed in the terminal. It is especially valuable in forensic investigations, as it provides a chronological record of user activity, including file creation, system modifications, and potential attempts to cover tracks.

Q5 : What program used the file didyouthinkwedmakeiteasy.jpg during its execution?

binwalk

By reviewing the .bash_history file, we found the command binwalk didyouthinkwedmakeiteasy.jpg, indicating that the program binwalk was used to analyze the file.

Binwalkis a forensic tool for inspecting binary files and firmware images, commonly used to extract embedded files, hidden data, or compressed archives within a file.

Q6 : What is the third goal from the checklist Karen created?

profit

By inspecting the Checklistfile in /root/Desktop/, we see that it contains a series of goals set by Karen. The third goal listed in the file is Profit.

Q7 : How many times was Apache run?

0

To determine how many times Apache was run, the /var/log/apache2/ directory was examined. All log files access.log, error.log, and other_vhosts_access.log are 0 KB in size, indicating that Apache was never run.

Q8 : This machine was used to launch an attack on another. Which file contains the evidence for this?

irZLAohL.jpeg

Under the /root/ directory, the image irZLAohL.jpeg was found, providing evidence that this machine was used to launch an attack. The image shows a Windows-based victim and a terminal window running flightSim from C:\Users\Bob\AppData\Local\Temp.

FlightSim is a tool for generating malicious network traffic, including port scans, brute-force attacks, and DNS tunneling. Its execution in the terminal, along with the presence of irZLAohL.jpeg, indicates that this machine was used to simulate or perform attacks on other systems.

Q9 : It is believed that Karen was taunting a fellow computer expert through a bash script within the Documents directory. Who was the expert that Karen was taunting?

Young

Upon examining the script, it includes a series of commands designed to display system information. The commands include:

  • Showing the current working directory (pwd).
  • Displaying the default network route using ip route.
  • Listing network connections associated with port 80 using netstat.

Toward the end of the script, there is a line that stands out as a taunt:

echo "Heck yeah! I can write bash too Young"

Q10 : A user executed the su command to gain root access multiple times at 11:26. Who was the user?

postgres

Analyzing the /var/log/auth.log file shows multiple entries at 11:26 where the su command was used to switch users. The logs indicate that KarenHacker successfully switched to the postgresaccount using root privileges. This confirms that the user executing su at 11:26 was postgres.

Q11 : Based on the bash history, what is the current working directory?

/root/Documents/myfirsthack/

Analyzing the .bash_history file shows that the user executed the command cd ../Documents/myfirsthack/. This indicates that the current working directory is /root/Documents/myfirsthack/.
